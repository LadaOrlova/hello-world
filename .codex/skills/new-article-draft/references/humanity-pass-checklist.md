# Humanity Pass Checklist

Пройти этот чеклист после первого полного черновика.

## 1) AI-smell паттерны (удалить/переписать)

- Универсальные заходы без фактуры: "В современном мире", "Важно отметить", "Нельзя недооценивать".
- Слишком ровный ритм: одинаковая длина предложений 8-15 слов по всему тексту.
- Абстрактные формулировки без актора: "было принято решение", "произошла оптимизация".
- Слишком гладкие переходы без конфликтов и trade-offs.
- Параграфы, которые можно переставить местами без потери смысла.

## 2) Human signal усилители (добавить)

- Конкретный субъект действия: кто именно решил, отказался, поменял подход.
- Контекст решения: когда, при каком ограничении, что стояло на кону.
- Интеллектуальное трение: где тезис может не сработать.
- Точечные маркеры живой речи: короткий вопрос, контраст, самокоррекция.

## 3) Ритм и читабельность

- В каждом разделе есть смесь коротких и длинных фраз.
- Нет перегруза номинализациями и канцеляритом.
- Подзаголовки несут мысль, а не декоративные labels.

## 4) Source integrity

- У каждого внешнего факта есть URL.
- У каждой прямой цитаты есть первоисточник.
- Нет claims вида "исследования показывают" без ссылки.

## 5) Самооценка Humanity Risk (0-100)

Считать по штрафам:

- +20: много шаблонных AI-оборотов.
- +20: текст слишком равномерный по ритму.
- +20: слабая конкретика (мало имен/чисел/контекста).
- +20: нет контраргументов/ограничений.
- +20: есть утверждения без источников.

Интерпретация:

- `0-20`: ок для публикации.
- `21-40`: нужен точечный rewrite.
- `41+`: нужен полный humanization pass по разделам.
