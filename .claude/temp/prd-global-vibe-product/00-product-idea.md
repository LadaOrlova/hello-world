# Идея продукта

## Настройки
- **Язык:** Русский
- **Аналитика:** PostHog

## Описание идеи

Глобальный англоязычный self-service продукт, который обучает:
1. **Основам создания продукта** (методология AJTBD, продуктовое мышление)
2. **Вайб-кодингу** (создание продуктов с помощью AI — Claude Code, Cursor и др.)

### Ключевая гипотеза
Сейчас идёт огромный тренд: люди начинают создавать продукты сами. Им нужна база по созданию продукта + быстрый вход в вайб-кодинг.

### Формат (гипотеза)
- Не просто записанные курсы, а **AI-консультант / гайд**, который ведёт пользователя по шагам
- Пошагово рассказывает что делать и почему именно это
- Self-service (не требует живого ведущего на каждом шагу)
- В идеале — интерактивный guidance, а не пассивный просмотр видео

### Существующие продукты автора (на русском рынке)
1. **BOOST Intensive** (boost-intensive.ru) — недельный интенсив по вайб-кодингу. Живые воркшопы, кураторы, от идеи до MVP за неделю. Цена $379-$1049. Рейтинг 9.6/10. 95% участников без тех. бэкграунда.
2. **Product How To** (zamesin.ru/producthowto/) — курс по AJTBD-методологии для продактов и предпринимателей. 11000+ выпускников, рейтинг 9.2/10.

### Рынок
- Глобальный, англоязычный (фокус на US, но не только)
- Тренд: демократизация создания продуктов через AI
