# Идея продукта (v2)

## Настройки
- **Язык:** Русский
- **Аналитика:** PostHog

## Описание идеи

AI-гайд по продуктовому мышлению + обучение вайбкодингу — гибрид курса и инструмента.

### Концепция
Продукт, в который ты заходишь, рассказываешь, что хочешь сделать. Он:
1. **Задаёт вопросы** — помогает написать продуктовые требования, выделить риски
2. **Обучает в процессе** — объясняет "почему мы это делаем", "как это работает" прямо по ходу работы
3. **Генерирует документы** — PRD, аналитический план, лендинг-копи, которые ты потом передаёшь в Lovable, Replit, Cursor или Claude Code
4. **Учит вайбкодингу** — практические микро-гайды: как установить Cursor, Claude Code, как с ними работать

### Ключевое отличие от v1 (LaunchPilot)
- НЕ генерирует код / НЕ деплоит продукт
- Учит тебя ДУМАТЬ о продукте (product thinking) и ДЕЛАТЬ самому (через внешние инструменты)
- Output = документы + знания, а НЕ готовый код

### Формула
Курс + инструмент = ты одновременно учишься И решаешь свою реальную задачу

### Целевой пользователь
Человек, который хочет что-то создать (продукт, бизнес), но не знает КАК — ни что строить (product thinking), ни как строить (vibe-coding tools).
