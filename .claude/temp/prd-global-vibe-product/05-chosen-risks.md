# Выбранные риски для учёта в PRD

## Учитываемые риски:

1. **Риск #1 (Score 10.0): Guided AI experience не даёт результата**
   - AI-гайд должен довести non-technical человека до работающего задеплоенного продукта
   - PRD должен заложить механики, которые минимизируют "застревание" пользователя

2. **Риск #2 (Score 7.5): Не готовы платить за self-service подписку**
   - Нужно доказать ценность в free tier до paywall
   - PRD должен заложить чёткий Aha-момент до оплаты
   - Рассмотреть альтернативные модели монетизации как fallback

3. **Риск #6 (Score 5.3): Retention будет низким**
   - Education-продукты известны низким retention
   - PRD должен заложить accountability-механики, progress tracking, причины возвращаться

## Не учитываемые в PRD (проверяются отдельно):
- Риск #7: Тренд vibe-coding (проверяется аналитикой трендов)
- Риск #4: Конкуренция (проверяется анализом конкурентов — Агент 1)
- Риск #3: Глобальный рынок (проверяется экспертными интервью)
- Остальные риски — проверяются по мере запуска
