# Идея продукта

## Настройки
- **Язык:** English (primary) + Russian (secondary market)
- **Аналитика:** PostHog

## Описание идеи

**Teen Boost** — недельный интенсив по вайбкодингу и AI-агентам для подростков (13-17 лет).

Адаптация существующего продукта BOOST (успешный интенсив для взрослых с NPS 9+/10, 4 потока, 120+ студентов на поток) под аудиторию подростков.

### Ключевые отличия от взрослого BOOST:
- Проекты адаптированы под интересы подростков (игры, мерч-магазины, AI-арт, контент-инструменты)
- Покупатель — родитель, пользователь — подросток (двойная аудитория)
- Международный фокус (US + Western Europe), вторичный рынок — Россия
- Оплата через Stripe (международный), отдельный лендинг для каждого рынка

### Что подростки изучают:
1. Настройка Claude Code и инструментов разработки
2. Создание своего MVP (веб-приложение) с помощью вайбкодинга
3. Работа с данными и исследование рынка
4. Создание скиллов и AI-агентов
5. Создание AI-продуктов с командами агентов
6. Деплой на сервер

### Бизнес-контекст:
- **Бизнес-задача:** Выход на новый сегмент (подростки) и новый рынок (международный)
- **Тип продукта:** Онлайн-курс / образовательный интенсив
- **Монетизация:** Разовая оплата за интенсив
- **Ограничения:** Маленькая команда (2-4 чел), vibe-coding подход
- **Данные:** 4 успешных потока BOOST для взрослых, NPS 9+/10, обратная связь студентов
- **Горизонт:** < 2 недели до запуска MVP
- **Каналы:** Paid Ads + Community / сарафан / соцсети
- **Конкуренты:** iD Tech, AI Camp, Code Ninjas (teens); нет прямых конкурентов в "vibe coding for teens"
