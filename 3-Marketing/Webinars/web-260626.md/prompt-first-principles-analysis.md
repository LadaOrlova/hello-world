## ЗАДАНИЕ

Ты — один из самых авторитетных стратегических аналитиков мира на стыке технологий, продуктового менеджмента и бизнес-стратегии. Твой уровень: Ben Thompson (Stratechery) + Clayton Christensen (Innovator's Dilemma) + Hamilton Helmer (7 Powers) + Marty Cagan (Inspired).

### Контекст

Февраль 2026. Мир пережил качественный скачок:
- AI-модели (Claude Opus, GPT-o3, Gemini Ultra) достигли уровня, где они пишут код, анализируют рынки, создают стратегии лучше среднего специалиста
- Стоимость написания кода упала до ~$0 (inference cost → 0, вайбкодинг, AI-агенты)
- Стоимость исследований упала до ~$0 (AI читает, синтезирует, анализирует)
- Появились AI-агенты с автономностью: могут сами действовать, не только думать (Claude Code, Codex, Devin)
- SaaS-индустрия потеряла $1+ трлн капитализации ("SAASpocalypse 2026")
- 30%+ кода в крупнейших компаниях мира пишется AI
- Появилась роль "Full Stack Builder" — один человек делает работу 5-10

### Метод: First Principles Analysis

Проведи анализ из первых принципов (Elon Musk / Aristotle method):

1. **Определи аксиомы** — что стало бесплатным или почти бесплатным:
   - Написание кода = ~$0
   - Исследования и анализ = ~$0
   - Когнитивный труд на задачу = ~$0
   - Координация = ~$0
   - Создание прототипа = ~$0
   - Генерация контента (текст, изображения, видео) = ~$0

2. **Построй логические цепочки**: если X стало бесплатным → что следует? → что следует из этого? → и ещё дальше? Иди до НЕОЧЕВИДНЫХ выводов — тех, которые не видны с первого взгляда.

3. **Выведи новые принципы создания продуктов** — правила, которые раньше не работали (или не существовали), а теперь стали фундаментально верными.

### Чего НЕ повторять

У нас уже есть 21 принцип. НЕ повторяй их. Вот список существующих:

**6 основных принципов вебинара:**
1. "Сначала строй, потом исследуй" — MVP дешевле исследования
2. "Вкус (taste) — bottleneck" — код = commodity, judgment = дефицит
3. "Любовь к делу = конкурентное преимущество" — одержимость = экономический моат
4. "1 человек + 20 агентов = бывший отдел" — интеллект + агентность
5. "Moat ≠ фича" — фичи копируются за дни
6. "PM = оператор фабрики гипотез" — 20 гипотез/нед, убивай 19

**15 дополнительных принципов:**
7. "Продукт = работник, а не инструмент" — outcome-based pricing
8. "Закон Кристенсена для AI: контекст > интеллект" — прибыль ушла в контекст
9. "Flywheel обучения бьёт flywheel данных" — скорость learning loop
10. "GPT-лестница: рынки открываются ступенями" — capability release dates
11. "Дизайн для двух аудиторий: человек + AI-агент" — Agent Experience (AX)
12. "Впервые за 60 лет новая парадигма интерфейса" — intent-based UI
13. "Продукт-для-одного: адаптивный UI" — фиксированный дизайн мёртв
14. "Провал бесплатен — побеждает скорость перебора" — fail fast literally free
15. "Дилемма инноватора ускорилась в 100x" — 10 месяцев, не 10 лет
16. "Экспертиза расщепилась: ответы = commodity, вопросы = дефицит" — unknown unknowns
17. "Build-Measure-Learn мёртв → Prompt-Test-Learn" — build исчез как этап
18. "Бремя доказательства перевернулось" — найм надо обосновывать, не автоматизацию
19. "Средний менеджмент испаряется → песочные часы" — AI-менеджеры вместо людей-менеджеров
20. "SaaS → SaS (Service-as-Software)" — TAM x15
21. "AI поедает посевной фонд" — нет джуниоров = нет сеньоров через 7 лет

### Критерии качества

- Каждый принцип должен быть **неочевидным** — не банальным следствием
- Каждый должен быть **проверяемым** — есть данные или логика, не мнение
- Каждый должен быть **полезным** — PM или фаундер может принять решение на его основе
- Каждый должен быть **уникальным** — не повторять 21 принцип выше
- Приоритет: глубина > ширина. Лучше 10 глубоких, чем 15 поверхностных
- Язык вывода: **русский**
