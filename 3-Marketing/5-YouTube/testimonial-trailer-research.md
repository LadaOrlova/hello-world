## 1. Структура видео-трейлера

### Общий принцип

Трейлер из отзывов участников — это НЕ набор последовательных интервью. Это **смонтированная история трансформации**, где реплики разных спикеров переплетаются в единый нарратив.

### Структура на 5-7 минут (оптимальная длина)

| Блок | Хронометраж | Содержание | Визуал |
|------|-------------|------------|--------|
| **HOOK (крючок)** | 0:00-0:10 | Самый мощный результат/цитата участника. "Я заработал первый миллион через 3 месяца после воркшопа" | Крупный план спикера + текстовый оверлей с цифрой |
| **БОЛЬ (проблема)** | 0:10-1:30 | 3-4 участника описывают свои проблемы ДО воркшопа. Каждый — 15-20 сек. | Чередование спикеров, приглушённая музыка, возможно ч/б или десатурация |
| **ТОЧКА ПЕРЕЛОМА** | 1:30-2:30 | Момент открытия / инсайта на воркшопе. "Когда я понял, что..." "Меня перевернуло..." | Фрагменты записей воркшопа, B-roll аудитории |
| **ТРАНСФОРМАЦИЯ** | 2:30-4:00 | Конкретные изменения в жизни/работе. Числа, факты, эмоции. | Энергичная музыка, motion graphics с цифрами |
| **РЕЗУЛЬТАТЫ** | 4:00-5:00 | Конкретные метрики и достижения. "Выручка выросла на X%", "Нашёл работу мечты" | Текстовые плашки с цифрами, графики |
| **SOCIAL PROOF** | 5:00-5:30 | Быстрая нарезка коротких фраз 5-7 участников: "Рекомендую", "Лучшее вложение" | Быстрый монтаж, 3-5 сек на фразу |
| **CTA (призыв)** | 5:30-6:00 | Финальная мощная цитата + экран с призывом к действию | Брендированный слайд, ссылка, QR-код |

### Структура на 3 минуты (сокращённая версия)

| Блок | Хронометраж | Содержание |
|------|-------------|------------|
| **HOOK** | 0:00-0:05 | Одна убойная цитата |
| **БОЛЬ** | 0:05-0:45 | 2-3 спикера, по 12-15 сек |
| **ПЕРЕЛОМ + ТРАНСФОРМАЦИЯ** | 0:45-1:45 | Объединённый блок, 3-4 спикера |
| **РЕЗУЛЬТАТЫ** | 1:45-2:30 | Числа и факты, быстрый монтаж |
| **CTA** | 2:30-3:00 | Финальная фраза + экран |

### Структура на 8-10 минут (развёрнутая версия)

Для длинной версии добавляются:
- **Контекст** (1:30-2:00): краткое объяснение формата воркшопа (можно голосом ведущего или текстовыми плашками)
- **Мини-истории**: 2-3 развёрнутые истории участников по 60-90 сек каждая (полная арка: проблема → воркшоп → результат)
- **Расширенный social proof**: больше коротких фрагментов от разных участников

## 2. Эмоциональная арка повествования

### Оптимальная эмоциональная кривая

```
Эмоциональный уровень
│
│         ★ Пик (Результат)
│        / \
│       /   \
│      /     ★ CTA (надежда)
│     / Подъём
│    /  (Трансформация)
│   /
│  ★ Дно (Боль)
│ /
│/
★ Интрига (Хук)
└──────────────────────── Время
```

### Пять эмоциональных фаз

**Фаза 1: ИНТРИГА (0-10 сек)**
- Эмоция: любопытство, удивление
- Приём: показать конечный результат в начале ("результат вперёд")
- Зритель думает: "Как?! Расскажите подробнее"

**Фаза 2: БОЛЬ / УЗНАВАНИЕ (10 сек - 1:30)**
- Эмоция: эмпатия, узнавание себя
- Критически важно: зритель должен ПОЧУВСТВОВАТЬ боль как свою собственную
- Приём: конкретные, сенсорные описания ("Каждое утро я открывал ноутбук с ощущением, что тону")
- Повтор боли от разных спикеров усиливает эффект — зритель видит, что проблема массовая

**Фаза 3: ОТКРЫТИЕ / ПЕРЕЛОМ (1:30-2:30)**
- Эмоция: надежда, "ага-момент"
- Приём: показать конкретный момент инсайта
- Музыка начинает светлеть, темп чуть ускоряется

**Фаза 4: ТРАНСФОРМАЦИЯ / РЕЗУЛЬТАТЫ (2:30-5:00)**
- Эмоция: вдохновение, воодушевление
- Приём: конкретные цифры и факты, произнесённые самими участниками (не текстом, а живым голосом)
- Motion graphics усиливают числа визуально
- Темп нарастает, музыка становится энергичнее

**Фаза 5: ПРИЗЫВ / НАДЕЖДА (5:00-6:00)**
- Эмоция: решимость, мотивация
- Приём: самая запоминающаяся фраза + чёткий CTA
- Финальная цитата должна быть рефлексией ("Жалею только об одном — что не сделал этого раньше")

### Ключевое правило эмоциональной арки

> Чем глубже зритель прочувствует БОЛЬ — тем мощнее будет эффект ТРАНСФОРМАЦИИ. Не бойтесь задержаться на проблемах. Без контраста нет истории.

## 3. Таргетинг по сегментам аудитории

### Принцип: "Я вижу себя в этом видео"

Каждый зритель должен найти в трейлере **персонажа-двойника** — человека с похожей ситуацией, проблемами и контекстом. Это называется "identification" (идентификация).

### Как встроить сегменты в одно видео

**Сегмент 1: Основатели / предприниматели**
- Боль: "Я всё делаю сам, продукт не масштабируется, команда не понимает задачи"
- Текстовый lower third: "Анна, основатель EdTech-стартапа"
- Результат: "Выручка выросла в 3 раза за полгода"

**Сегмент 2: Владельцы бизнеса**
- Боль: "Бизнес упёрся в потолок, не понимаю, что делать дальше"
- Lower third: "Дмитрий, владелец производственной компании"
- Результат: "Запустил новое направление, которое приносит 40% выручки"

**Сегмент 3: Наёмные менеджеры**
- Боль: "Не могу доказать ценность своей работы руководству, проекты буксуют"
- Lower third: "Мария, продакт-менеджер в финтехе"
- Результат: "Получила повышение через 2 месяца после воркшопа"

**Сегмент 4: Переход в своё дело**
- Боль: "Хочу уйти из найма, но страшно, нет плана"
- Lower third: "Сергей, бывший корпоративный менеджер"
- Результат: "Запустил свой бизнес, первые клиенты через месяц"

**Сегмент 5: Переход в продакт-менеджмент**
- Боль: "Хочу стать продактом, но не знаю с чего начать, нет портфолио"
- Lower third: "Елена, бывший маркетолог"
- Результат: "Получила оффер продакт-менеджера в IT-компании"

**Сегмент 6: Занятые люди / работающие мамы**
- Боль: "Нет времени на учёбу, разрываюсь между работой и семьёй"
- Lower third: "Ольга, мама двоих детей, руководитель отдела"
- Результат: "Воркшоп уместился в мой график, а результаты превзошли ожидания"

### Правила размещения сегментов в видео

1. **Блок "Боль"**: здесь КАЖДЫЙ сегмент должен быть представлен хотя бы одной репликой (по 10-15 сек). Это критически важный момент идентификации
2. **Блок "Трансформация"**: повторить 3-4 ключевых сегмента с конкретными результатами
3. **Social proof**: быстрая нарезка — разнообразие лиц и контекстов максимально
4. **Lower thirds обязательны**: имя, должность/роль, компания — это то, что позволяет зрителю опознать "своего"
5. **Порядок сегментов**: начинать с самого массового сегмента (ваш основной ICP), заканчивать блоком — самым эмоциональным

## 4. Техники монтажа

### 4.1. Ритм и темп

| Блок видео | Средняя длина кадра | Темп монтажа | Тип переходов |
|------------|---------------------|--------------|---------------|
| Хук | 2-3 сек | Быстрый | Жёсткие склейки (hard cuts) |
| Боль | 4-6 сек | Медленный | Мягкие переходы, dissolve |
| Перелом | 3-4 сек | Средний → быстрый | Жёсткие склейки |
| Трансформация | 3-5 сек | Нарастающий | Жёсткие склейки + motion graphics |
| Результаты | 2-3 сек | Быстрый | Быстрые склейки, jump cuts |
| Social proof | 2-3 сек | Очень быстрый | Jump cuts |
| CTA | 5-8 сек | Медленный (пауза) | Fade to branded screen |

### 4.2. Правило 3-5 секунд

> Исследования показывают, что зрителю нужно примерно 3 секунды, чтобы полностью воспринять кадр. Если кадр длится больше 5 секунд — внимание начинает угасать. (Motion Edits)

Поэтому: каждые 3-5 секунд на экране должно происходить что-то визуально новое:
- Смена ракурса
- Вставка B-roll
- Появление текстового оверлея
- Смена спикера

### 4.3. Переключение между спикерами

**Оптимальная частота:**
- В блоке "Боль": менять спикера каждые **15-20 секунд**
- В блоке "Трансформация": каждые **20-30 секунд** (дать рассказать историю)
- В блоке "Social proof": каждые **3-5 секунд** (быстрая нарезка коротких фраз)
- Общее правило: **не более 30-40 секунд одного спикера подряд** без B-roll или визуальной перебивки

**Техника "стыковки" спикеров:**
- Один спикер начинает мысль, другой продолжает ту же идею другими словами
- Это создаёт эффект "хора голосов" — массовое подтверждение

### 4.4. B-roll и перебивки

Что использовать как B-roll из записей воркшопа:
- Лица участников во время работы (крупный план, сосредоточенность)
- Руки, пишущие в тетрадях / на стикерах
- Экран с презентацией (размытый фон)
- Общий план аудитории
- Реакции (смех, кивки, "ага-моменты")
- Аплодисменты в конце

**Правило B-roll**: каждые 10-15 секунд "говорящей головы" — вставлять 2-4 секунды B-roll. Голос спикера продолжает звучать поверх B-roll (voice-over).

### 4.5. Текстовые оверлеи и lower thirds

**Lower thirds (плашки с именем спикера):**
- Появляются в первые 2-3 секунды каждого нового спикера
- Остаются на экране 4-6 секунд
- Формат: Имя (крупный шрифт, bold) + роль/должность (мелкий, light)
- Шрифт: sans-serif (Arial, Montserrat, Inter)
- Полупрозрачный фон под текстом для читаемости

**Ключевые цитаты на экране:**
- Самые мощные фразы дублировать текстом на экране
- Использовать для просмотра без звука (85% видео в соцсетях смотрят без звука)
- Максимум 8-10 слов на экране одновременно
- Анимация: плавное появление (fade in), не резкое

**Цифры и результаты:**
- Выносить в крупные анимированные плашки
- Пример: "+147% выручки" — большим шрифтом, с анимацией счётчика
- Появляется синхронно с моментом, когда спикер произносит число

### 4.6. Музыка

**Общие правила:**
- Фоновая музыка идёт на протяжении ВСЕГО видео
- Громкость: -18 to -24 dB под голосом спикера (голос всегда доминирует)
- Музыка без вокала (инструментальная)

**Подбор по блокам:**

| Блок | Темп (BPM) | Характер | Жанр |
|------|-----------|----------|------|
| Хук | 100-120 | Энергичный, интригующий | Cinematic, electronic |
| Боль | 60-80 | Минорный, задумчивый | Ambient piano, strings |
| Перелом | 80-100 | Нарастающий, с подъёмом | Build-up, crescendo |
| Трансформация | 100-120 | Мажорный, вдохновляющий | Uplifting corporate, indie |
| Результаты | 110-130 | Триумфальный, энергичный | Motivational, cinematic |
| CTA | 90-100 | Уверенный, завершающий | Corporate positive |

**Музыкальные акценты:**
- Drop (удар) на момент перехода от "Боли" к "Перелому"
- Нарастание (rise/swell) перед кульминационными цитатами
- Короткая тишина (0.5-1 сек) перед самой мощной фразой — усиливает эффект

**Где брать музыку:**
- Artlist.io, Epidemic Sound, Musicbed (платные, высокое качество)
- YouTube Audio Library, Pixabay Music (бесплатные)
- Envato Elements (по подписке)

### 4.7. Цветокоррекция

- **Блок "Боль"**: слегка приглушённые тона, лёгкая десатурация (не ч/б, но менее яркие)
- **Блок "Трансформация" и далее**: насыщенные, тёплые тона, хороший контраст
- Контраст между блоками визуально подчёркивает трансформацию
- Единый LUT / фильтр для всех спикеров, чтобы видео выглядело цельным

## 5. Хук и удержание внимания

### Первые 5 секунд решают всё

> Данные Wistia: падение зрителей максимально в первые 5 секунд. Сильный хук удерживает 70% зрителей после 10 секунд. Слабый — теряет больше половины.

### 7 типов хуков (от лучшего к худшему для тестимониал-трейлера)

**1. Результат вперёд (лучший вариант)**
> Спикер: "За 3 месяца после воркшопа моя выручка выросла на 200%"
- Почему работает: конкретное число + конкретный срок = интрига "как?"

**2. Провокационное заявление**
> Спикер: "Я потратил 5 лет на MBA. Этот воркшоп дал мне больше за 2 дня"
- Почему работает: контраст + провокация + конкретика

**3. Эмоциональная цитата**
> Спикер (эмоционально): "Это буквально изменило мою жизнь. Не преувеличиваю"
- Почему работает: искренняя эмоция цепляет мгновенно

**4. Вопрос к зрителю (текстом на экране)**
> Текст: "Что, если за 2 дня можно понять то, на что уходят годы?"
- Плюс: вовлекает. Минус: менее конкретно.

**5. Статистика**
> Текст/голос: "87% участников получили повышение или запустили бизнес в течение 6 месяцев"
- Почему работает: социальное доказательство в числах

**6. Быстрый монтаж фраз**
> 5 спикеров по 1 секунде: "Лучшее решение" / "Перевернуло всё" / "Наконец понял" / "Не ожидал такого эффекта" / "Рекомендую всем"
- Почему работает: энергия, ритм, массовость

**7. Визуальный контраст**
> Экран делится пополам: слева — уставшее лицо "до", справа — улыбка "после"
- Работает как дополнение к любому из вышеописанных хуков

### Правило тройного хука

Для максимального удержания все три канала должны работать одновременно в первые 5 секунд:
- **Визуальный хук**: динамичное движение, крупный план, яркий кадр
- **Аудио хук**: мощная фраза спикера или драматический музыкальный удар
- **Текстовый хук**: оверлей с ключевой цифрой или провокационным утверждением (критично для просмотра без звука)

### Удержание внимания на протяжении видео

1. **Каждые 30-60 секунд — "мини-хук"**: новый поворот, неожиданная цифра, эмоциональный момент
2. **Варьировать темп**: не держать один ритм монтажа — чередовать быстрые и медленные участки
3. **Open loops**: намёки на то, что будет дальше ("Но самое интересное произошло потом..." — и переключение на другого спикера)
4. **Pattern interrupt**: каждые 45-60 сек визуально менять стиль (добавить анимацию, сменить фон, показать B-roll)

## 6. Примеры лучших трейлеров

### Примеры для изучения и вдохновения

**1. Coursera — компиляция отзывов студентов**
- Формат: 30-секундные нарезки из отзывов разных студентов
- Сила: разнообразие лиц и историй, быстрый монтаж, эмоциональные фразы
- Структура: Боль → Обучение → Результат за 30 секунд

**2. Codecademy — история карьерной трансформации**
- Формат: 2-3 минутная история одного студента
- Сила: полная арка трансформации (из медицины в IT за год)
- Приём: конкретная, детальная история с числами и фактами

**3. Scrimba — истории студентов буткемпа**
- Формат: серия отдельных видео-отзывов
- Сила: каждый студент — отдельная персона (каждый сегмент аудитории видит "своего")
- Приём: lower thirds с именем и новой должностью после обучения

**4. Udemy — промо-трейлер курса**
- Рекомендуемая Udemy структура: 2 минуты максимум
- Формат: инструктор + нарезка отзывов + демонстрация материала
- Приём: начать с брендированного слайда → 1 фраза о себе → ценность → отзывы → CTA

**5. Masterclass — трейлеры курсов**
- Формат: 1-2 минуты, кинематографическое качество
- Сила: высокое качество картинки, эмоциональная музыка, минимум текста
- Приём: акцент на личности эксперта + атмосфера

**6. Tony Robbins — трейлеры мероприятий**
- Формат: 3-5 минут, нарезка из разных мероприятий
- Сила: энергия зала, крупные планы эмоций, контрастный монтаж
- Приём: хук с самым эмоциональным моментом → быстрая нарезка → развёрнутые истории → CTA

### Что взять из каждого примера

| Приём | Откуда | Как адаптировать |
|-------|--------|-----------------|
| Быстрая нарезка коротких фраз | Coursera | Блок "Social proof" — 5-7 участников по 3-5 сек |
| Полная арка одного человека | Codecademy | Мини-история в блоке "Трансформация" — 60-90 сек |
| Персона в lower third | Scrimba | Каждый спикер = конкретная роль и контекст |
| Открытие с брендом | Udemy | Первые 2 сек — лого/название воркшопа |
| Кинематографическое качество | Masterclass | Цветокоррекция, музыка, ритм монтажа |
| Энергия зала | Tony Robbins | B-roll с записей: лица, реакции, аплодисменты |

## 7. Чек-лист для видеоредактора

### Пре-продакшн (до монтажа)

- [ ] Отсмотреть ВСЕ записи воркшопа и отзывов
- [ ] Выписать таймкоды лучших цитат с указанием: кто говорит, о чём, эмоциональность (1-10)
- [ ] Разметить цитаты по блокам: Боль / Перелом / Трансформация / Результат / Social proof
- [ ] Разметить цитаты по сегментам аудитории (фаундеры, наёмные, карьерный переход и т.д.)
- [ ] Выбрать 8-12 ключевых спикеров (разнообразие по полу, возрасту, роли)
- [ ] Найти B-roll фрагменты из записей воркшопа (лица, реакции, работа в группах)
- [ ] Подобрать 2-3 музыкальных трека (минорный для боли, мажорный для трансформации)

### Монтаж

- [ ] **Хук**: первые 5 сек — самый мощный результат + текстовый оверлей
- [ ] **Блок "Боль"**: 3-5 спикеров, по 15-20 сек, разные сегменты аудитории
- [ ] **Перебивки**: B-roll каждые 10-15 сек "говорящей головы"
- [ ] **Блок "Перелом"**: музыкальный переход, смена настроения
- [ ] **Блок "Трансформация"**: 3-4 развёрнутых отзыва по 20-30 сек
- [ ] **Числа на экране**: каждое упомянутое число — текстовый оверлей с анимацией
- [ ] **Social proof**: быстрая нарезка 5-7 коротких фраз по 3-5 сек
- [ ] **CTA**: финальная мощная цитата + экран с призывом (5-8 сек)
- [ ] **Субтитры/оверлеи**: ключевые фразы продублированы текстом
- [ ] **Lower thirds**: каждый спикер подписан (имя + роль), 4-6 сек на экране
- [ ] **Музыка**: идёт непрерывно, громкость -18/-24 dB под голосом
- [ ] **Тишина**: 0.5-1 сек тишины перед самыми ударными фразами
- [ ] **Общая длина**: 5-7 минут (оптимум) или 3 минуты (сокращённая версия)

### Пост-продакшн

- [ ] Цветокоррекция: единый стиль для всех спикеров
- [ ] Цветовой контраст: приглушённые тона в "Боли" → насыщенные в "Трансформации"
- [ ] Аудио: нормализация голосов (одинаковая громкость всех спикеров)
- [ ] Аудио: удаление шумов, эха, фоновых звуков
- [ ] Финальный экран: лого + URL + QR-код + текст CTA ("Записаться", "Узнать подробнее")
- [ ] Экспорт: 1920x1080 для YouTube/лендинга + 1080x1080 или 1080x1920 для соцсетей
- [ ] Нарезка коротких клипов: 3-5 отдельных клипов по 15-60 сек для соцсетей

### Технические параметры для FFmpeg / автоматизации

```bash
# Нормализация аудио (loudnorm)
ffmpeg -i input.mp4 -af loudnorm=I=-16:TP=-1.5:LRA=11 output.mp4

# Добавление субтитров из SRT-файла
ffmpeg -i input.mp4 -vf "subtitles=subs.srt:force_style='FontName=Montserrat,FontSize=24,PrimaryColour=&HFFFFFF,OutlineColour=&H000000,Outline=2'" output.mp4

# Склейка клипов из файла списка
ffmpeg -f concat -safe 0 -i clips.txt -c copy output.mp4

# Добавление фоновой музыки с duck под голос
ffmpeg -i video.mp4 -i music.mp3 -filter_complex "[1:a]volume=0.15[music];[0:a][music]amix=inputs=2:duration=first" -c:v copy output.mp4

# Fade in/out на видео и аудио (последние 3 секунды)
ffmpeg -i input.mp4 -vf "fade=t=in:st=0:d=1,fade=t=out:st=357:d=3" -af "afade=t=in:st=0:d=1,afade=t=out:st=357:d=3" output.mp4

# Экспорт для соцсетей (квадрат 1080x1080)
ffmpeg -i input.mp4 -vf "crop=ih:ih,scale=1080:1080" -c:a copy output_square.mp4

# Экспорт для Stories/Reels (вертикаль 1080x1920)
ffmpeg -i input.mp4 -vf "crop=ih*9/16:ih,scale=1080:1920" -c:a copy output_vertical.mp4
```
