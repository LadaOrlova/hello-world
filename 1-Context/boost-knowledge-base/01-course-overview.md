# BOOST: обзор интенсива

## Суть

**BOOST** — интенсив по созданию продуктов с AI (вайб-кодинг). Учит создавать полноценные продукты с использованием нейросетей (Cursor, Claude Code) за одну неделю — **от идеи до работающего MVP на своём сервере**. Опыт программирования не требуется.

Позиционирование: **"За одну неделю — от идеи до работающего продукта на своём сервере"**

**95% участников — без техбэкграунда** (продакты, маркетологи, предприниматели, бизнес-аналитики).
