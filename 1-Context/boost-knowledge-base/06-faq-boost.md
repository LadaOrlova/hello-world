## О BOOST

### Что такое интенсив BOOST?
Интенсив по вайбкодингу и созданию продуктов с нейросетями. Полный цикл: анализ рынка → PRD → работа с данными → вайбкодинг. Цель — создавать продукты в 10 раз быстрее. Сайт: https://boost-intensive.ru/

### Кто ведёт BOOST?
Ваня Замесин и Илья Измайлов (автор канала Vibe Living, основатель buyerapp).

### Для кого подходит BOOST?
1. Фаундеры — быстро проверять гипотезы
2. Владельцы бизнеса — автоматизировать процессы
3. Переход из найма в своё
4. Переход в продукт — портфолио для резюме
5. Продакты — понять AI на практике
95% участников не умеют программировать.

### Нужно ли уметь программировать?
Нет. 95% участников — без техбэкграунда. Продакты, маркетологи, предприниматели, бизнес-аналитики. Claude Code пишет код — участник объясняет словами, что нужно.

### Это курс для программистов?
Нет. Это курс для создателей продуктов — фаундеров, продактов, владельцев бизнеса. Вы не станете профессиональным программистом. Вы освоите навыки, которые УСИЛЯТ ваши скиллы десятикратно.

### Если я умею веб-кодить, будет ли мне полезно участие?
Если делал на Lovable, Replit — участие будет ценно. Если уже самостоятельно пишет код, деплоит через Claude Code, использует скиллы агентов — для такого участника интенсив не будет ценным.

### Чем отличается от других курсов по вайб-кодингу?
- Другие курсы: 70-150k руб, 2-6 месяцев, учебные проекты
- BOOST: 39 900 руб, неделя, ВАША идея + AJTBD-методология + AURA
- Уникально: AJTBD-методология (понимание клиента) + AI (создание продукта)

### Можно ли работать над рабочей задачей?
Да. Многие участники работали над рабочими проектами: Стас автоматизировал процессы автосервиса, Олег создал чат-бот по банковской документации, Александр выстроил пайплайн подготовки контента.

### На каком уровне нужно знать AI?
На любом. Начинают с нуля. Участники приходят с разным опытом — от "никогда не пробовал" до "активно использую ChatGPT". Каждый найдёт ценность на своём уровне.

## Инструменты

### Что за инструменты используются?
- **VSCode** — рекомендуемый код-редактор для работы с Claude Code (бесплатный)
- **Claude Code** — AI-агент, который может работать с файлами и запускать команды на компьютере (как ChatGPT, но с доступом к файлам)
- **Wispr Flow** — голосовой ввод
- **OpenRouter** — доступ к AI API для создания ботов
- **Next.js, Tailwind, PostgreSQL** — стек для создания MVP

### Почему Claude Code, а не Cursor?
- Claude Code даёт на порядок больше токенов за те же деньги — тупо дешевле
- Claude Code даёт одну из лучших моделей Opus 4.5/4.6 на рынке
- Научившись работать с Claude Code, сможете перейти на Codex CLI и использовать любые инструменты
- Рекомендуем устанавливать Claude Code в VSCode (вместо Cursor) — это бесплатно и работает так же

### Что нужно для прохождения BOOST?
- VSCode (бесплатно) — рекомендуется вместо Cursor
- Claude Code Max ($100/мес или $200/мес подписка)
- VPN (если в России)
- 1-3 идеи для работы
- Инструкцию пришлют после оплаты. Помощь с установкой и оплатой из России входит в курс.

Гайды по установке:
- [Установка VSCode на macOS](https://aura-framework.ru/guides/ustanovka-vs-code-na-mac)
- [Установка VSCode на Windows](https://aura-framework.ru/guides/ustanovka-vs-code-na-windows)
- [Установка расширения Claude Code в VSCode](https://aura-framework.ru/guides/ustanovka-rasshireniya-claude-code-v-vs-code)

### Какой VPN рекомендуется для оплаты и обучения из России?
Рекомендуемые VPN: Red Shield, Browsec, Amnezia VPN. При оплате зарубежных сервисов важно, чтобы страна выпуска карты и страна VPN совпадали. Гайд по оплате: https://aura-framework.ru/guides/payments-foreign-services

### Сколько стоит подписка на Claude Code?
Для прохождения интенсива потребуется подписка Claude Code Max за $100/мес. Это не входит в стоимость интенсива. Помогут с оплатой из России — подскажут проверенные сервисы.

### Входит ли стоимость Claude Code в стоимость BOOST?
Нет. Подписки на Claude Code и другие сервисы оплачиваются самостоятельно.

## Формат и расписание

### Сколько длится интенсив?
1 неделя: 7 воркшопов (включая установочный). Воркшоп 0 — 60 минут, остальные — по 150 минут.

### Даты ближайших потоков BOOST?
- **Поток 3:** 7–15 марта 2026
- **Поток 4:** 20–28 марта 2026 (распродан)
- **Поток 5:** даты уточняются, следите за обновлениями на сайте https://boost-intensive.ru/
7 воркшопов на каждом потоке (6 основных + 1 установочный).

### Программа BOOST (пример расписания)
- Воркшоп 0 (60 мин) — Установка Claude Code и VSCode
- Воркшоп 1 (150 мин) — PRD и первый лендинг с Claude Code
- Воркшоп 2 (150 мин) — Прототип и деплой продукта
- Воркшоп 3 (150 мин) — Исследование рынка с AI
- Воркшоп 4 (150 мин) — Автоматизация, MCP, транскрипция
- Воркшоп 5 (150 мин) — UX, QA-тестирование, доработка MVP
- Воркшоп 6 (150 мин) — Деплой в продакшн, итоги

Точное расписание с датами и временем публикуется на сайте https://boost-intensive.ru/ для каждого потока.

### Куда деплоят проекты?
Каждому участнику выдаётся VPS (через Telegram-бота) на 2 недели. Все необходимые инструменты и инструкции предоставляются на воркшопе.

### А если не успею за неделю?
Записи доступны навсегда. В тарифе "С поддержкой" — 3 недели помощи после интенсива.

### Когда получу записи BOOST?
Тариф "Записи" — записи первого потока сразу после оплаты. Если купил интенсив — записи до старта потока не отправляют.

## AURA и Makers

### Что такое AURA?
AI-платформа на основе методологии AJTBD (Advanced Jobs To Be Done). Инструменты для анализа рынка, сегментации, построения графа работ.

### Что включено в подписку AURA?
- **Maker (в тарифе "Интенсив"):** 2 месяца, AI-инструменты для продактов (ценность 8 000 ₽)
- **Founder (в тарифе "С поддержкой"):** 3 месяца, все AI-инструменты + кредиты + созвоны с Ваней (ценность 60 000 ₽)

### Что такое Makers?
Сообщество вокруг AURA — создатели продуктов, фаундеры, продакты.

### Как продлить доступ к AURA после окончания подписки с курса?
Привязать карту — подписка продлится автоматически: https://aura-framework.ru/profile/subscription. Также можно вручную купить кредиты: https://aura-framework.ru/usage.

### Когда начисляются кредиты AURA при покупке BOOST?
При покупке BOOST студент получает месяцы подписки AURA (не кредиты напрямую). Подписка активируется с момента регистрации на платформе AURA. Кредиты начисляются помесячно и имеют срок жизни — с даты начисления до этого же числа следующего месяца. Неиспользованные подписочные кредиты сгорают. Купленные вручную кредиты не сгорают.

### Есть ли скидки на BOOST для студентов КДП?
Нет. BOOST — отдельный продукт с максимально лояльными ценами, скидки не предоставляются. Скидка 20% за повторное прохождение действует только на курс «Как делать продукт».

### Куда я попаду после BOOST?
Все, кто купил тариф "Записи" или прошёл интенсив, попадают в чат и канал выпускников (Boost Alumni Club).

### Я прохожу курс «Как делать продукт» — получу ли полный доступ к BOOST?
Нет. В рамках КДП (с потока 58) включены 1-3 базовых воркшопа из BOOST. Для полного интенсива нужно приобрести BOOST отдельно.
