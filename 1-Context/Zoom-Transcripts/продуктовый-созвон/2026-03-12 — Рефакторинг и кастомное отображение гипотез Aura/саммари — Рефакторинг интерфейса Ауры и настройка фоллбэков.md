# [Командный-созвон] Рефакторинг интерфейса Ауры и настройка фоллбэков
**Дата**: 12 марта 2026

---

Вот структурированное саммари командного созвона на основе предоставленных тезисов.

## Краткое содержание
Встреча была посвящена техническому обновлению сервиса «Аура» (рефакторинг, миграция пользователей, биллинг) и обсуждению архитектуры новой системы ZIS. Ключевым административным решением стала отмена и перенос конференции из-за нехватки ресурсов, а также фокус на продажах 59-го потока курса через вебинары.

## Статусы и обновления
- **Илья Изма (Аура):** Проводит рефакторинг шаблонов (переход на HTML), внедрил цепочку фоллбэков (Gemini -> Qwen -> Grok -> Kimy) для отказоустойчивости и готовит миграцию пользователей в новую версию с полноценным биллингом.
- **Ivan Zamesin (Проектирование):** Выявил расхождение в расходах на API (OpenRouter списывает $100/день, админка показывает меньше), работает над архитектурой ZIS и таск-трекера.
- **Alena Mins (Маркетинг/Биллинг):** Контролирует оплаты в старой Ауре, готовит анонсы для пользователей Бусти/курсов и единый CTA для вебинара.
- **Руслан Жу (Продажи):** Сообщил о листе ожидания на 60-й поток (19 человек) и высокой эффективности живых продаж через Zoom/Телемост.
- **Lada Orlova (Операционка):** Подготовила прогноз охватов по конференциям, отметила расхождение реальных задач с планом в Notion.
- **Maxim Kurepov (DevOps):** Настраивает дев-окружение для ботов и занимается пробросом UTM-меток.

## Ключевые решения
- **Перенос конференции:** Отмена мероприятия через две недели и перенос на неопределенный срок. Решение принято Иваном Замесиным из-за дефицита личного ресурса и необходимости подготовить новый курс.
- **Миграция в новую Ауру:** После завершения работы над шаблонами Илья Изма вывесит плашку о переезде в старой версии.
- **Доступ для Бусти:** Участникам 5-го Бусти решено дать доступ к Ауре на 4 месяца и 1500 кредитов.
- **Работа с корпоративными клиентами:** Уралсибу (14 человек) разрешен вход на 59-й поток по условию постоплаты.
- **Оплата инфраструктуры:** GitHub будет оплачен Настей Дяченко с новой корпоративной карты, репозитории переносятся в Zamesin Team.
- **Архитектура ZIS:** В проект закладывается возможность расширения до таск-трекера и системы бонусов.
