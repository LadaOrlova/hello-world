## Краткое содержание
Созвон посвящен определению и структурированию заданий для предстоящего воркшопа по работе с AI агентами. Основной акцент сделан на разработке прикладных задач для продуктов и предпринимателей, исключая чистое кодирование, а также на использовании команд агентов для улучшения продуктовых требований (PRD) и создании чат-бота.

## Ключевые решения
- **Ориентировать задачи воркшопа на прикладные сценарии для продуктов и предпринимателей:** Отказаться от кодинга в качестве основной задачи для агентов, сосредоточившись на понятных бизнес-задачах, таких как наем сотрудников или доработка продуктовых требований.
- **Основная задача для Agent Teams:** Разработка и доработка продуктовых требований (PRD) с использованием команды агентов.
- **Включить создание Telegram-бота с AI и обработкой фотографий** как одну из задач воркшопа (первая половина).
- **Включить навык "создание и использование Agent Teams"** для студентов воркшопа.
- **Структура воркшопа:** Первая половина будет посвящена созданию бота, вторая — использованию команды агентов для работы с PRD.
- **Иван Земесин составит четкое объяснение**, когда, в каких ситуациях и для каких задач полезны Agent Teams.

## Основные темы обсуждения
- Целевая аудитория воркшопа (продукты и предприниматели) и выбор соответствующих прикладных задач для AI агентов.
- Обсуждение использования AI агентов для кодинга против использования для более абстрактных бизнес-задач (PRD, наем).
- Конкретные примеры прикладных задач для агентов (наем сотрудника, доработка PRD).

- Структурирование и наполнение воркшопа, включая порядок и вариативность заданий.
