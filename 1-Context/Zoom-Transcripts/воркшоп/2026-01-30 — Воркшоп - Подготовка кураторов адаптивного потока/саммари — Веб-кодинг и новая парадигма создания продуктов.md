## Ключевые концепции

### 1. Вайб-кодинг (Новая парадигма)
- **Суть идеи**: Это способ создания продуктов, где фокус смещается с написания кода на управление ИИ-инструментами. Границы возможного расширяются в процессе: через пару месяцев вы начнете делать вещи, о которых даже не думали на старте.
- **Почему это важно**: Традиционный аутсорс — это долго и дорого. Вайб-кодинг позволяет одному человеку без навыков программирования за 2 месяца сделать то, что команда делала полгода.
- **Ключевой пример**: Илья Изма сэкономил несколько миллионов рублей, создав микро-стартап (бот калорий), приносящий $500 в месяц, без найма разработчиков.

### 2. Снижение когнитивной нагрузки через UI
- **Суть идеи**: Интерфейс инструментов разработки (Cursor) часто выглядит пугающе («хакерски»), что вызывает сопротивление мозга. Светлая тема и упрощение панелей делают среду визуально похожей на привычный Notion.
- **Почему это важно**: Психологический барьер «я не программист» мешает обучению. Упрощение интерфейса (Light mode) помогает новичкам не «посыпаться» на первых этапах.
- **Ключевой пример**: Решение Ивана Замесина переключиться на светлую тему прямо во время демонстрации после обратной связи от участников.

### 3. Обучение через «грязные» эксперименты
- **Суть идеи**: Начинать нужно с азов и «пробования ногами», а не с настройки сложных систем вроде GitHub или глубоких параметров API. Важно завести человека в цикл «попробовал — ошиблось — починил».
- **Почему это важно**: Глубокая теория на старте перегружает. Использование прямого API нейросетей слишком дорого ($1000+), поэтому эффективнее использовать готовые интерфейсы подписок.
- **Ключевой пример**: Критика гайда Жени Васева за излишнюю сложность («перемудрил») и переход к упрощенной схеме настройки через «три точки».

### 4. Экономика соло-разработки
- **Суть идеи**: Стоимость входа в вайб-кодинг состоит из подписок на правильные сервисы (около $100 за Claude/Cursor), что несопоставимо с затратами на штат разработчиков.
- **Почему это важно**: Понимание реальных цифр (комиссии, курсы валют) позволяет планировать бюджет проекта.
- **Ключевой пример**: Сравнение затрат Ильи Изма (от 500к до 1 млн руб. в месяц на аутсорс) против стоимости подписки на ИИ.

## Фреймворки и инструменты
- **Cursor**: Основная IDE (среда разработки) с ИИ. Используется как главный хаб для написания кода. Важно настроить Layout (Agent Sidebar) и тему.
- **Claude / Claude Code**: Инструмент от Anthropic. Claude Code — терминальный помощник для глубокой работы с кодом.
- **WhisperFlow**: Инструмент, упомянутый в цепочке установки для работы с аудио/текстом.
- **Zamesin Sandbox (Яндекс Облако)**: Серверная среда для тестирования и размещения проектов.
- **Сервисы оплаты («За Рубль», «Плати рублём»)**: Инструменты для обхода ограничений при оплате зарубежных подписок из РФ.

## Практические упражнения
- **Настройка окружения**: Участники проходили цепочку: VPN $\rightarrow$ Регистрация/Оплата Claude ($100) $\rightarrow$ Установка Claude Code $\rightarrow$ WhisperFlow $\rightarrow$ Cursor.
- **Конфигурация интерфейса**: Переключение Cursor в светлую тему и настройка расположения панелей файлов.
