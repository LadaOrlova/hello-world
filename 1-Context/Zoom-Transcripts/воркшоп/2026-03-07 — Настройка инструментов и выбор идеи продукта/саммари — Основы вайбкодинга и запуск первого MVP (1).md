## Ключевые концепции

### 1. Вайбкодинг и «Иллюзия горы»
- **Суть:** Новая парадигма программирования, где акцент смещается с написания синтаксиса на управление инструментами ИИ и фантазию. Главная преграда для новичка — психологическое сопротивление («иллюзия горы»), которое нужно преодолеть через быстрые первые успехи.
- **Почему это важно:** Инструменты позволяют реализовать концепт за 10 минут (например, во время завтрака), превращая программирование в творческий процесс.
- **Ключевой пример:** Спикер 1 навайбкодил новый концепт обучения за 10 минут, просто используя свои идеи и ИИ.

### 2. Агентность Claude Code и управление контекстом
- **Суть:** В отличие от чат-ботов, Claude Code обладает агентностью — он может самостоятельно читать/писать файлы и запускать команды в терминале. Однако при достижении лимита контекстного окна (200 000 токенов) модель может «тупеть» (галлюцинировать).
- **Почему это важно:** Правильное управление контекстом (очистка чата перед новым этапом) сохраняет остроту «ума» модели.
- **Ключевой пример:** Рекомендация нажимать «плюс» (новый чат) при переходе от создания PRD к написанию кода лендинга или MVP.

### 3. Точность PRD как залог качественного кода
- **Суть:** ИИ не ленится прописывать детали, поэтому он лучше человека подходит для создания объемных продуктовых требований (PRD).
- **Почему это важно:** Качество и точность реализации кода напрямую зависят от того, насколько детально и точно составлен документ PRD.
- **Ключевой пример:** Проведение PRD-интервью с Клодом через WhisperFlow для извлечения максимума деталей из идеи пользователя.

### 4. Биоинформатика и ИИ в науке
- **Суть:** Использование вычислительных методов для расшифровки «языка ДНК». Нейросети (AlphaFold) произвели революцию, сокращая время исследования белков с лет до минут.
- **Почему это важно:** Демонстрирует, что ИИ меняет не только IT-сектор, но и фундаментальную науку и медицину.
- **Ключевой пример:** Упоминание модели AlphaFold, совершившей прорыв в предсказании структур белков.

## Фреймворки и инструменты
- **Claude Code**: Основной агентный инструмент. Выгоднее Cursor (больше токенов за ту же подписку). Использовать режим `Edit automatically`.
- **Модели Opus 3.5 (4.6) и Sonnet 4.6**: Opus — для логики и PRD (в 5 раз дороже), Sonnet — для простых текстовых задач.
- **JTBD (Jobs To Be Done)**: Методология, использованная для написания текста лендинга, фокусирующаяся на задачах пользователя.
- **WhisperFlow**: Инструмент для проведения PRD-интервью голосом/текстом.
- **FeatureDev**: Плагин, устанавливаемый перед началом разработки MVP.
- **Amazing Boost Server Bot**: Бот для получения доступов к VPS-серверам (хостинг Una).
