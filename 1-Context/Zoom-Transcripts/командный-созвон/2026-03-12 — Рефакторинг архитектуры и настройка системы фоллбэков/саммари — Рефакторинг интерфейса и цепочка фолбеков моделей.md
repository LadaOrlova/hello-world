# [Командный-созвон] Рефакторинг интерфейса и цепочка фолбеков моделей
**Дата**: 12 марта 2026

---

Вот структурированное саммари по итогам созвона:

## Краткое содержание
Встреча была посвящена техническому рефакторингу проекта «Аура» и оптимизации расходов на API (OpenRouter). Также обсуждались операционные задачи по образовательным потокам (59-й и 60-й), отмена/перенос конференции из-за нехватки ресурсов и проектирование внутренней системы ЗИС.

## Статусы и обновления
- **Илья Изма (Аура)**: Проводит масштабный рефакторинг (интерфейс временно свернут), внедряет HTML-шаблоны для гипотез и цепочку фолбеков (Gemini -> Qwen -> Grok -> Kimi) для экономии бюджета.
- **Ivan Zamesin**: Задачи 10-го спринта не выполнены (перенос на 11-й), статья для HackerNoon готова частично.
- **Руслан Жу (Продажи)**: На 59-й поток набрано 137 человек (10,5 млн руб. чистыми). Формируется лист ожидания на 60-й поток (уже 20 человек, включая «Уралсиб»).
- **Alena Mins (Маркетинг)**: В процессе поиск спикеров на следующую неделю; по «Boost 5» осталось всего 2 места из 12.
- **Lada Orlova (Продвижение)**: Подготовлен план посевов по конференциям; на сегодняшний вебинар зарегистрировано 430 человек.
- **Maxim Kurepov (IT-инфраструктура)**: Подтвердил проблемы с оплатой GitHub Teams; готов настраивать дев-окружение для ботов.

## Ключевые решения
- **Перенос конференции (Ivan Zamesin)**: Conf переносится на неопределенный срок, так как за 2 недели подготовить качественный продукт невозможно.
- **Оптимизация «Ауры» (Илья Изма)**: Внедрить в админку разбивку расходов по дням для синхронизации с данными OpenRouter.
- **Условия для «Буста» (Alena Mins / Ivan Zamesin)**: Пользователям на поддержке предоставить доступ на 4 месяца и по 1500 кредитов в месяц.
- **Стратегия по 60-му потоку (Ivan Zamesin)**: Официально не анонсировать, максимально переводить желающих на 59-й поток (возможна постоплата).
- **GitHub (Ivan Zamesin / Maxim Kurepov)**: Перенести репозиторий в организацию `Zamesin Team` и оплатить задолженность новой картой.
- **Уведомления (Ivan Zamesin / Алина Ков)**: Отказаться от ручных постов в каналах об окончании подписки; это должен делать только бот индивидуально.
