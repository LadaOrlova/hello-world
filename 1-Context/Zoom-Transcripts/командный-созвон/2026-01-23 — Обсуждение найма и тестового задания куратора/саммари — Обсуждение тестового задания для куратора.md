Вот структурированное саммари командного созвона на основе предоставленных тезисов:

## Краткое содержание
Встреча была посвящена организации процесса найма 5 кураторов для курса. Основное внимание уделили критериям отбора (владение Claude Code и субагентами), структуре тестовых заданий, системе оплаты и распределению ролей в команде отбора.

## Статусы и обновления
- **Alena Mins**: В процессе создания Typeform для сбора заявок (необходимо добавить 11 вопросов и прогнать через нейросеть для проверки полноты).
- **Илья Изма**: Определены технические критерии отбора (умение работать с субагентами и CLI-инструментами).
- **Ivan Zamesin**: Сформулирована методологическая база (фокус на JTBD и Claude Code) и финансовая модель оплаты кураторов.

## Ключевые решения
- **Стек и методология**: Основным инструментом выбран Claude Code (модель Opus) как самый мощный и универсальный на рынке. Все выпускники должны работать по методологии Jobs To Be Done (Ivan Zamesin).
- **Процесс отбора**: Утверждена трехэтапная воронка: 1. Анкета + тестовое в Typeform; 2. 15-минутный созвон на адекватность; 3. Техническое интервью с Ильей (Ivan/Alena).
- **Тестовые задания**: Сделать 3 варианта лендинга Boost (проверка работы с субагентами) и **обязательное** написание гайда по Claude Code (проверка логики мышления) (Ivan/Илья).
- **Условия работы**: Оплата 40 000 руб. (фикс) + 15 000 руб. (премия за оценку 4+) за поток (~25 часов работы). Дополнительно: доступ к платформе Aura, тариф «Мейкер» и сообщество.
- **Каналы поиска**: Начинать с внутренних чатов (выпускники, мейкеры), при нехватке — выходить в большой канал (Ivan).

## Другое (Контекст и расчеты)
- **Экономика**: Стоимость часа куратора составит около 2500 руб. (при выполнении KPI). Маржинальность на кураторов заложена на уровне 17-20%.
- **Тайминг**: Курс включает 7 воркшопов по 2-2.5 часа. Нагрузка на одного куратора — 5 студентов.
- **Разногласия**: Была дискуссия об объеме ТЗ (сократили с 30 до 3 вариантов лендинга) и обязательности гайда (Алёна предлагала сделать опциональным, Иван настоял на обязательности для проверки мышления).
- **Инструментарий**: Отмечено, что 100$ на токены Claude Code должно хватить для большинства задач. Claude Code признан универсальной базой для обучения любым CLI-инструментам.
