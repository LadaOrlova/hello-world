# [Консультация] Оформление Azure кредитов стартапам
**Дата**: 12 февраля 2026

---

## О клиенте
Клиент (Спикер 2) — представитель стартапа, зарегистрированного в США, заинтересованный в получении облачных ресурсов и использовании передовых AI-моделей (таких как GPT-4o) для своих проектов.

## Запрос клиента
Клиент обратился за инструкциями по получению бесплатных или льготных кредитов Microsoft Azure для стартапов, а также по доступу и развёртыванию AI-моделей через платформу Azure для снижения затрат на инфраструктуру и разработку.

## Ключевые рекомендации

### 1. Получение начальных Azure кредитов ($1000)
-   **Суть рекомендации:** Зарегистрироваться на специальном сайте для стартапов, используя Gmail, и верифицировать свой аккаунт через LinkedIn. Это позволит получить первые $1000 Azure кредитов, которые действуют в течение трёх месяцев.
-   **Обоснование:** Это первый и самый быстрый способ получить доступ к основным облачным сервисам Azure и начать тестирование, не дожидаясь полной верификации компании.

### 2. Получение дополнительных Azure кредитов ($5000) для стартапа
-   **Суть рекомендации:** После получения начальных кредитов и ввода данных о компании, пройти процесс верификации стартапа на платформе. Это включает предоставление юридического адреса компании и загрузку подтверждающих документов (например, свидетельства о регистрации ИП/компании), которые должны соответствовать указанным данным.
-   **Обоснование:** Успешная верификация стартапа открывает доступ к дополнительным $5000 кредитов, что значительно расширяет возможности для разработки, масштабирования и долгосрочного использования облачных ресурсов. В случае проблем с верификацией рекомендуется обращаться в службу поддержки.

### 3. Доступ и использование AI-моделей через Azure AI Foundry
-   **Суть рекомендации:** Использовать платформу Azure AI Foundry (ранее Microsoft AI Studio) для изучения каталога доступных AI-моделей (включая самые обновлённые, такие как GPT-4o) и выбора тех, которые подходят для проекта.
-   **Обоснование:** AI Foundry предоставляет централизованный и удобный интерфейс для работы с передовыми AI-моделями, позволяя легко развёртывать их и интегрировать в свои приложения, используя полученные Azure кредиты.

### 4. Запрос квоты на использование AI-моделей
-   **Суть рекомендации:** Для полноценного использования выбранных AI-моделей (например, GPT-4o) необходимо запросить квоту. Это делается через раздел "Manage Code" в Azure AI Foundry, где нужно выбрать модель, регион и заполнить специальную форму запроса.
-   **Обоснование:** Квота необходима для выделения вычислительных ресурсов под конкретную AI-модель и разрешения на трату кредитов на её использование. Правильное и детальное заполнение формы повышает шансы на быстрое одобрение квоты в течение нескольких дней.

## Следующие шаги для клиента
- [ ] Зарегистрироваться на сайте Microsoft для стартапов (ссылка предоставлена в ТГ) и выполнить Sign Up.
- [ ] Верифицировать аккаунт через LinkedIn для получения первых $1000 Azure кредитов.
- [ ] При регистрации ввести юридический адрес своей компании, который соответствует её документам.
- [ ] Привязать банковскую карту к аккаунту Azure (желательно использовать карту, на которой нет значительных средств, как мера предосторожности).
- [ ] Инициировать процесс верификации стартапа на платформе для получения дополнительных $5000 кредитов, загрузив все необходимые подтверждающие документы.
- [ ] Изучить платформу Azure AI Foundry и доступные AI-модели (например, GPT-4o).
- [ ] Отправить запрос на получение квоты для выбранных AI-моделей через раздел "Manage Code" в Azure AI Foundry, заполнив форму.
- [ ] В случае возникновения проблем с верификацией стартапа или получением квоты, связаться с поддержкой Azure.

## Инсайты и выводы
*   Microsoft активно привлекает стартапы, предоставляя значительные объемы Azure кредитов, что является мощным инструментом для снижения затрат на инфраструктуру и тестирование AI-решений.
*   Процесс получения кредитов и доступа к AI-моделям является многоступенчатым и требует внимательности к деталям: от верификации аккаунта/компании до запроса квот на конкретные модели.
*   Платформа Azure AI Foundry предоставляет стартапам прямой доступ к передовым AI-моделям, позволяя быстро интегрировать их в свои продукты и экспериментировать с новейшими технологиями.
*   Эффективное использование кредитов возможно только после получения квоты на конкретную AI-модель, что является критически важным шагом в процессе настройки.
