## Паттерны успеха

### 1. Сегментация по работам > демографическая
Во всех кейсах переход от демографической сегментации ("возраст", "доход", "размер компании") к сегментации по работам дал прорывные результаты.

### 2. Фокус на одном сегменте
Компании, которые сфокусировались на одном сегменте вместо "всё для всех", показали лучшие результаты:
- LiFT: эксперты → топ-менеджеры
- ArtDent: все стоматологические услуги → "ремонт за день"
- Legal Resources: все клиенты → топ-20

### 3. Интервью = основа решений
Медианное количество интервью: 15-25. Ни один кейс не обошёлся без качественных интервью.

### 4. Быстрый результат от коммуникации
Самый быстрый ROI — изменение маркетинговых сообщений по результатам интервью:
- Neginski: лендинг за неделю → ROMI 2000%
- ЭКОПСИ: переформулировка ценности → возврат 30% клиентов

### 5. "Выход на предыдущую/высокоуровневую работу" — самые мощные механики
Кейсы с наибольшим ростом используют эти механики:
- LiFT (x3 чек): высокоуровневая работа
- Rossko (+90 млн): предыдущая работа
- SKAI (+75%): предыдущая работа
- Pleada (+20%): предыдущая работа

### 6. Пивот через отрицательный результат
Отрицательный результат исследования — тоже результат:
- Дрон-шоу: маркетплейс не сработал → прибыльное агентство
- Forma+: неправильный сегмент → правильный → x2.5
