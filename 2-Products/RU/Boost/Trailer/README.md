## Что сделано

1. Скопированы записи завершающих воркшопов потоков 2, 3, 4 (в `Final-Workshops/`)
2. Расшифровки прочитаны, извлечены 100+ цитат
3. Индивидуальные аудиотреки (29 спикеров) транскрибированы через mlx-whisper
4. 240 кандидатных клипов нарезаны из 3 видеофайлов
5. Каждый клип оценён через OpenRouter (Gemini Flash) по visual/energy/audio/content
6. Лучший клип от каждого спикера выбран автоматически
7. Финальный трейлер собран с нормализацией звука (loudnorm -16 LUFS)

## Что можно улучшить

1. **Точность нарезки** — текущие клипы 15 секунд (целые окна), в идеале обрезать до конкретных фраз 5-12 секунд
2. **Субтитры** — добавить русские субтитры для просмотра без звука
3. **Музыка** — добавить фоновую музыку с нарастанием к финалу
4. **Титры** — добавить имя и роль каждого спикера (lower third)
5. **Заставка и CTA** — добавить титульную карточку BOOST в начале и URL в конце
