# Примеры вопросов для подготовки к экзамену

> Эти вопросы помогут понять формат экзамена. На экзамене будут другие вопросы, но того же типа и сложности.

---

## Формат экзамена

| Часть | Время | Описание |
|-------|-------|----------|
| Теория | 40 мин | 25 вопросов: тесты, открытые ответы, мини-кейсы, «найди ошибку» |
| Практика | 2 часа | AI-кейс: решаете бизнес-задачу с помощью чат-бота, который знает все данные кейса |

## Примеры теоретических вопросов

### Пример 1: Multiple choice

**Какой из принципов лежит в основе AJTBD?**

- A) Клиент всегда знает, чего хочет
- B) Человек действует для выполнения работ, которые мозг формулирует для удовлетворения потребностей
- C) Продукт определяет поведение клиента
- D) Рынок определяется демографией клиентов

### Пример 7: Мини-кейс

**Кейс**: Вы запускаете новый продукт — онлайн-платформу для изучения шахмат. У вас есть 3 гипотезы:
1. Рынок онлайн-обучения шахматам достаточно большой
2. Сегмент «родители, которые хотят развить ребёнка» готов платить 3000 ₽/мес
3. Вы сможете привлекать клиентов через YouTube-канал

**Задание**: Расставьте гипотезы по приоритету проверки, используя RAT. Объясните логику.

## Формат практической части

### Что будет на экзамене

1. Вы получите бизнес-кейс (B2C-компания) с конкретной бизнес-задачей
2. AI-бот будет знать все данные кейса: метрики, интервью, сегментацию, конкурентов, юнит-экономику
3. Вы сможете задавать боту вопросы о данных (но он НЕ будет давать готовых решений)
4. Ваш ответ — структурированный текст из 5 блоков:
   - Граф работ целевого сегмента
   - ABCDX-сегментация
   - Механики создания ценности
   - RAT-гипотеза и эксперимент
   - One-liner и предложения по лендингу

### Что оценивается

- Правильное применение фреймворков AJTBD к конкретной ситуации
- Конкретность и обоснованность решений (не абстракции)
- Использование данных кейса (цитаты клиентов, метрики)
- Логическая связь между блоками ответа
- Связь решений с бизнес-задачей

### Советы по подготовке

1. Повторите все алгоритмы бизнес-задач (конверсия, ретеншн, чек, отток)
2. Потренируйтесь строить граф работ на реальных примерах
3. Выучите 16+ механик создания ценности
4. Убедитесь, что знаете формулу RAT и умеете ранжировать предположения
5. Потренируйтесь писать one-liner по формуле [Что] + [Core Job] + [Ценность]
6. Перечитайте главы книги курса про сегментацию, ценность и коммуникацию
