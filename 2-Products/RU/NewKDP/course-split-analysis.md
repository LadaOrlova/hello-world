# Анализ разрезки курса КДП на несколько продуктов

> Дата: 2026-03-15
> Источники: транскрипции 40 лекций + 12 воркшопов, CSV 1254 студентов (потоки 55-58), фидбэк 626 студентов (потоки 53-57), базы знаний KDP и BOOST, методология AURA/AJTBD, веб-ресерч по вайб-кодингу и AI-агентам

## Часть 1. Сегменты клиентов по работам (Jobs-to-Be-Done)

На основе анализа 1254 записей студентов и 626 входных анкет выделяются **6 ключевых сегментов**:

### Сегмент A: "Продакты за системой" (30%, ~235 чел. из оплаченных)

| Параметр | Значение |
|----------|----------|
| **Кто** | Middle/Senior/Head PM в IT-компаниях |
| **Опыт** | 1-4 года (пик) |
| **Big Job** | "Хочу знания + есть бизнес-задачи, но знания важнее" |
| **Core Job** | Систематизировать хаотичные знания в единую методологию |
| **Personal Job** | Побороть синдром самозванца, уверенность в решениях |
| **Триггер** | Ощущение "все решаю на чуйке", коллеги знают больше |
| **Тариф** | Максимум (48%) |
| **Удовлетворенность** | 8.4/10 |
| **Средний чек** | ~90K RUB |
| **Канал** | Telegram Вани (40%), рекомендация коллег (30%) |

**Экономическая привлекательность: ВЫСОКАЯ**
- Крупнейший сегмент по объему
- Высокий средний чек (берут Максимум)
- Низкий CAC (органика + сарафан)
- Высокий retention (возвращаются за продвинутыми тарифами)

---

### Сегмент B: "Собственники бизнеса" (18%, ~141 чел.)

| Параметр | Значение |
|----------|----------|
| **Кто** | Владельцы бизнеса, CEO, фаундеры |
| **Опыт** | Биполярный: 8+ лет (28%) или нет продуктового опыта (16%) |
| **Big Job** | "Запустить новый продукт на основе исследования, а не на чуйке" |
| **Core Job** | Получить алгоритм запуска, перестать сжигать деньги на невалидированные идеи |
| **Personal Job** | Больше зарабатывать, кайфовать от работы |
| **Триггер** | Обжёгся на запуске "на чуйке", потерял деньги |
| **Тариф** | Максимум (42%) + Максимум с фидбеком Вани (20%) |
| **Удовлетворенность** | 8.5/10 |
| **Средний чек** | ~110K RUB (готовы платить за фидбек) |
| **Канал** | Telegram (35%), Красинский (15%), рекомендации (25%) |

**Экономическая привлекательность: ОЧЕНЬ ВЫСОКАЯ**
- Максимальный средний чек (берут дорогие тарифы + фидбек)
- Платят сами (не ждут корпоративного бюджета)
- Высокая мотивация (на кону свои деньги)
- Потенциал upsell в BOOST (хотят быстро прототипировать)
- **Ограничение:** меньший объем сегмента

---

### Сегмент C: "Переходники в продукт" (28%, ~220 чел.)

| Параметр | Значение |
|----------|----------|
| **Кто** | Проект-менеджеры, бизнес-аналитики, маркетологи, дизайнеры |
| **Опыт** | 0-2 года или нет опыта (73%) |
| **Big Job** | "Хочу перейти в продукт / получить продуктовую базу" |
| **Core Job** | Получить фундамент: что такое продукт, как создавать ценность |
| **Personal Job** | Сменить профессию, больше зарабатывать, самозванец |
| **Триггер** | Хочу перейти в PM, но не знаю с чего начать |
| **Тариф** | Максимум (34%), База (24%) |
| **Удовлетворенность** | 8.6/10 (выше среднего — курс даёт то, чего у них нет) |
| **Средний чек** | ~75K RUB |
| **Канал** | Telegram (30%), коллеги-PM (35%), YouTube (15%) |

**Экономическая привлекательность: СРЕДНЯЯ**
- Большой объем сегмента
- Ниже средний чек (часть берут Базу)
- Высокая удовлетворенность = хороший NPS
- **Ограничение:** менее целевой для продвинутого контента, часть контента курса "пролетает мимо"

---

### Сегмент D: "Будущие предприниматели" (7%, ~55 чел.)

| Параметр | Значение |
|----------|----------|
| **Кто** | В найме, но мечтают о своём бизнесе |
| **Опыт** | Нет опыта (46%) или < 1 года (20%) |
| **Big Job** | "Хочу успешно запустить проект или понять, что не нужно" |
| **Core Job** | Алгоритм запуска + валидация идеи до увольнения |
| **Personal Job** | Больше зарабатывать, нетворк, понять "моё ли это" |
| **Тариф** | Максимум (56%) + Максимум с фидбеком (15%) |
| **Удовлетворенность** | 8.6/10 |
| **Средний чек** | ~95K RUB |
| **Канал** | Telegram (40%), бесплатная лекция (20%) |

**Экономическая привлекательность: СРЕДНЕ-ВЫСОКАЯ**
- Высокий средний чек (мотивированы, берут Максимум)
- Маленький объем
- Высокий потенциал cross-sell в BOOST (хотят прототипировать)
- **Ограничение:** малый объем, потребность сильно пересекается с собственниками

---

### Сегмент E: "Продакты за метриками" (5%, ~39 чел.)

| Параметр | Значение |
|----------|----------|
| **Кто** | Middle/Senior PM с конкретной бизнес-задачей |
| **Опыт** | 3-4 года (37%) |
| **Big Job** | "Продукт не продается / метрики не растут" |
| **Core Job** | Найти конкретную точку роста: конверсия, retention, ARPU |
| **Personal Job** | Уверенность в решениях, карьерный рост |
| **Тариф** | Практика (60%) + Стандарт+ (37%) |
| **Удовлетворенность** | 8.6/10 |
| **Средний чек** | ~70K RUB |
| **Канал** | Рекомендация коллег (50%) |

**Экономическая привлекательность: СРЕДНЯЯ**
- Самый маленький сегмент
- Ценят практику больше теории
- Высокий потенциал для узкого "интенсив по метрикам"

---

### Сегмент F: "Корпоративные покупатели" (10% по каналу привлечения, ~78 чел.)

| Параметр | Значение |
|----------|----------|
| **Кто** | Сотрудники Яндекс, Сбер, Т-Банк, Авито, МТС, Mindbox |
| **Big Job** | "Компания хочет внедрить продуктовый подход" |
| **Core Job** | Единый язык в команде, методология для всех |
| **Personal Job** | Выполнить KPI по обучению |
| **Тариф** | Разный (оплачивает компания) |
| **Средний чек** | Потенциально x5-10 (групповые покупки) |

**Экономическая привлекательность: ОЧЕНЬ ВЫСОКАЯ (недоиспользованная)**
- Высокий LTV (компания покупает на 3-10 сотрудников)
- Низкий CAC (1 продажа = 5-10 студентов)
- Бюджеты на обучение = стабильный доход
- **Ограничение:** требует B2B-продажного процесса, отдельного лендинга

## Часть 3. Предложение по разрезке курса

### Текущая проблема

Курс КДП — это один 2-месячный курс (27 лекций + 12 воркшопов) за ~80-110K руб., который пытается решить работы 6 разных сегментов одновременно. Это классический "всё для всех = ничего для никого".

**Конкретные симптомы:**
- Удовлетворенность 8.4-8.6 (хорошая, но не 9.5+)
- 40% на выходе "понимают как применить, но пока не используют"
- 9% "пока не понимают, как применить"
- Переходники скучают на стратегии, продакты скучают на основах

### Предлагаемая разрезка: 7 курсов

---

#### Курс 1: "Основы AJTBD: Как понять клиента" (ФУНДАМЕНТ)
**Для кого:** Сегменты C (переходники) + новички из всех сегментов
**Формат:** 2 недели, 8-10 лекций + 3 воркшопа
**Цена:** 25-35K RUB

**Содержание:**
- Что такое работы (Jobs) и почему они — причина всего
- Структура описания работы (контекст → триггер → результат)
- Типы работ: Big Job, Core Job, Small Jobs
- Граф работ — как строить и читать
- AJTBD-интервью: как проводить, демо B2C/B2B
- Практика: провести 3 интервью, построить граф
- 4 силы прогресса

**Адаптация под AI-эру:**
- Автотранскрипция интервью через Wispr Flow
- AI-экстракция работ из интервью (Claude Code скилл)
- Автопостроение графа работ по результатам интервью
- AI-модератор интервью (подсказки в реальном времени)

---

#### Курс 2: "Сегментация и выбор фокуса" (СТРАТЕГИЯ)
**Для кого:** Сегменты A (продакты) + B (собственники)
**Пререквизит:** Курс 1 или эквивалентные знания AJTBD
**Формат:** 1-2 недели, 5-6 лекций + 2 воркшопа
**Цена:** 30-40K RUB

**Содержание:**
- Настоящая vs ложная сегментация
- Сегментация по схожести графов работ
- ABCDX-сегментация существующих клиентов
- 10 факторов выбора фокусного сегмента
- Количественная валидация JTBD-исследования
- Упражнение "сегмент/не сегмент"

**Адаптация под AI-эру:**
- AI-агент для автоматической кластеризации клиентов по работам
- Автоматизированный ABCDX-анализ из CRM/данных
- AI-генерация гипотез о сегментах по данным из Aura

---

#### Курс 3: "Запуск нового продукта" (LAUNCH)
**Для кого:** Сегменты B (собственники) + D (будущие предприниматели)
**Пререквизит:** Курс 1
**Формат:** 2-3 недели, 6-8 лекций + 3 воркшопа
**Цена:** 40-55K RUB

**Содержание:**
- Product-Market Fit: стадии, диагностика
- Risk Assumption Test (RAT)
- Трекшн-карта
- MVP как "зонд для проверки рискованных предположений"
- Механика пивота
- Как найти сегмент для нового продукта
- Go-To-Market стратегии

**Адаптация под AI-эру:**
- Вайб-кодинг: студент создаёт работающий MVP за 2 дня (из BOOST WS1-2)
- AI-генерация PRD из результатов исследования
- Автодеплой на Vercel/Netlify
- AI-анализ рынка и конкурентов
- UX-тестирование прототипов AI-агентами

---

#### Курс 4: "Рост продукта: Метрики и стратегии" (GROWTH)
**Для кого:** Сегменты A (продакты) + E (продакты за метриками)
**Пререквизит:** Курс 1 + Курс 2
**Формат:** 2 недели, 6-8 лекций + 2-3 воркшопа
**Цена:** 35-50K RUB

**Содержание:**
- Юнит-экономика (с Красинским или самостоятельный модуль)
- Как развивать существующий продукт
- Локальный vs глобальный оптимум
- 80+ стратегий роста по бизнес-задачам
- Алгоритмы: снижение оттока, рост retention, рост ARPU
- Инноваторская дилемма и граф работ

**Адаптация под AI-эру:**
- AI-анализ воронки и поиск точек роста
- Автоматическая генерация гипотез роста из данных
- AI-агент для мониторинга метрик
- Дашборды аналитики через вайб-кодинг

---

#### Курс 5: "Коммуникация продукта" (COMMUNICATION)
**Для кого:** Все сегменты, особенно маркетологи (самая высокая удовлетворенность — 9.5!)
**Пререквизит:** Курс 1
**Формат:** 1 неделя, 4-5 лекций + 2 воркшопа
**Цена:** 20-30K RUB

**Содержание:**
- 7 факторов, определяющих покупку
- Ванлайнеры: формула "Что + Core Job + Ценность"
- Написание текста лендинга
- Формулы ценностных предложений
- Рост конверсии: aha-moment, триггеры, барьеры
- Как продать новое непривычное решение

**Адаптация под AI-эру:**
- AI-генерация ванлайнеров и ценностных предложений по сегментам
- Автосоздание лендинга через вайб-кодинг (из BOOST WS1)
- A/B тестирование копирайта AI-агентами
- Генерация рекламных креативов для Яндекс.Директ/VK/Telegram

---

#### Курс 6: "BOOST: Вайб-кодинг для продактов" (BUILD)
**Для кого:** Все сегменты — горизонтальный навык
**Формат:** 1 неделя, 7 воркшопов (~16 часов)
**Цена:** 30-80K RUB (уже существует)

**Содержание:** (уже есть)
- Настройка среды: VSCode, Claude Code, Wispr Flow
- PRD + лендинг
- MVP + деплой
- Анализ данных, ABCD-сегментация
- Субагенты, скиллы, автоматизация
- Telegram-боты, AI-продукты
- Демо-день

**Позиционирование:** "Свобода от зависимости от разработчиков"

---

#### Курс 7: "AURA: Продуктовая стратегия" (MASTERY)
**Для кого:** Сегменты A (Senior PM) + B (собственники) + F (корпоративные)
**Пререквизит:** Курсы 1-2 + опыт работы
**Формат:** 2-3 недели, masterclass
**Цена:** 50-80K RUB

**Содержание:**
- AURA как мета-фреймворк (AJTBD + UE + RAT + ABCDX)
- Синхронизация функций компании
- Главный алгоритм: от бизнес-задачи к продуктовому решению
- Стратегия масштабирования
- Стратегия выхода из конкуренции
- B2B-специфика AJTBD

**Адаптация под AI-эру:**
- Aura MCP — AI-агент для стратегических решений
- Автоматический конкурентный анализ
- AI-генерация стратегических гипотез

## Часть 4. Экономическая модель разрезки

### Текущая модель (один КДП):
- ~200 студентов/поток × ~80K = **~16M RUB/поток**
- 4-5 потоков/год = **~65-80M RUB/год**
- 1 продукт, 1 воронка, 1 лендинг

### Новая модель (7 курсов):

| Курс | Цена | Студ./мес. | Выручка/мес. |
|------|------|------------|--------------|
| 1. Основы AJTBD | 30K | 80-120 | 2.4-3.6M |
| 2. Сегментация | 35K | 40-60 | 1.4-2.1M |
| 3. Запуск продукта | 45K | 50-80 | 2.3-3.6M |
| 4. Рост продукта | 40K | 30-50 | 1.2-2.0M |
| 5. Коммуникация | 25K | 40-60 | 1.0-1.5M |
| 6. BOOST (есть) | 50K | 30-50 | 1.5-2.5M |
| 7. AURA Стратегия | 65K | 20-30 | 1.3-2.0M |
| **Итого** | | **290-450** | **11.1-17.3M/мес.** |

**Годовой потенциал: 130-210M RUB** (x1.6-2.6 от текущего)

### Механики роста:
1. **Лестница продуктов:** Курс 1 (30K) → Курс 2/3/5 (25-45K) → Курс 4/7 (40-65K) → BOOST (50K)
2. **Бандлы:** "Запуск нового продукта" = Курс 1 + 3 + 6 за 90K (скидка от 125K по отдельности)
3. **Корпоративный пакет:** Курс 1 + 2 + 4 для команды 5-10 чел. за 250-400K
4. **Подписка:** Доступ ко всем курсам за 15-20K/мес.

## Часть 5. Адаптация под вайб-кодинг и AI-агентов

### Контекст рынка (веб-ресерч)

**Ключевые тренды (конец 2025 — начало 2026):**
- "Vibe coding" — слово 2025 года по версии Collins Dictionary
- 92% разработчиков в США используют AI-кодинг ежедневно
- Cursor: $1.2B ARR, рост x11 за год
- Lovable: $200M ARR, рост x50 за год
- 25% стартапов YC имеют кодовую базу на 95% сгенерированную AI
- Andrew Ng: "Продуктовое управление — новый боттлнек, не инженерия"
- GoPractice: "PM = строитель, а не координатор"

**Ключевой инсайт:** JTBD + вайб-кодинг — уникальная комбинация на рынке. Никто в мире не объединяет глубокую продуктовую методологию (AJTBD) с практическим вайб-кодингом. Это конкурентное преимущество.

### Как адаптировать каждый курс

**Принцип:** В каждом курсе студент не просто учит теорию, а **строит работающий инструмент** через AI/вайб-кодинг.

| Курс | Что студент строит с AI |
|------|------------------------|
| 1. Основы AJTBD | AI-скилл для автоанализа интервью + граф работ в коде |
| 2. Сегментация | AI-агент для кластеризации клиентов + автоABCDX |
| 3. Запуск продукта | Работающий MVP + лендинг + деплой |
| 4. Рост продукта | Дашборд аналитики + AI-генератор гипотез роста |
| 5. Коммуникация | Лендинг + рекламные креативы + email-цепочки |
| 6. BOOST | Полный продукт с нуля за неделю |
| 7. AURA Стратегия | Стратегический AI-агент (Aura MCP) для компании |

### Новый горячий сегмент: "Переучивание профессий"

**Гипотеза из запроса подтверждается рынком:**
- Gartner: к 2026 году 70% новых приложений будут на low/no-code
- 56% премия к зарплате за AI-навыки
- HBR (февраль 2026): компаниям нужны "agent managers"

**Новые сегменты для разрезанных курсов:**

| Новый сегмент | Работа | Какой курс покупает |
|---------------|--------|---------------------|
| Маркетологи, осваивающие AI | "Хочу сам делать лендинги и тестить гипотезы, не ждать разработку" | Курс 5 + BOOST |
