#### По вопросу "ограниченный vs бессрочный доступ"

Это ключевое решение. Вот аргументы:

**За ограниченный (3-4 недели + 2 недели на добивание = 5-6 недель):**
- Дедлайн = forcing function. Self-paced без дедлайна = 10-15% доходимость. С дедлайном = 40-60%
- Создаёт когорту: студенты идут вместе, могут общаться, кураторы знают расписание
- Проще планировать нагрузку кураторов
- Создаёт upsell-момент: "курс закончился, хочешь продолжить → КДП Полный / Boost Pro"
- Паттерн рынка: GoPractice, Яндекс.Практикум, Ваня текущий — все с дедлайнами

**За бессрочный:**
- Проще продавать: "учись в своём темпе"
- Нет обязательств по кураторам после окончания
- Выпускники могут возвращаться к материалам (лояльность)

**Рекомендация:** Ограниченный доступ к кураторам (5-6 недель) + бессрочный доступ к записям и материалам. Куратор уходит, контент остаётся. Это даёт и forcing function для активной фазы, и "не потеряю доступ" для лояльности.

**Справка:** GoPractice работает полностью бессрочно — проходишь как удобно, рекомендуют по часу в день, без принуждения. Это другая модель: у них нет кураторов, только симулятор. У нас кураторы есть — и именно их доступ логично ограничивать по времени.

## Критический путь (что блокирует всё остальное)

```
Разведка (стадия 0)
    ↓
Утвердить программу с Ваней (1.3)  ← БЛОКЕР #1
    ↓
Решить: срок доступа (1.1) + тарифы (1.2)
    ↓
    ┌──────────┬──────────────┐
    ↓          ↓              ↓
Ваня пишет   Нарезка BOOST  Промпты
контент(2.1) онбординг(2.2) задания(2.4)
    ↓          ↓              ↓
    └──────────┴──────────────┘
              ↓
       Сборка шаблона (3.1)   ← БЛОКЕР #2
              ↓
       Тест: Илья проходит (5.1)
              ↓
       Бета-тест (5.3)
              ↓
       Открытие продаж (6.1)  ← БЛОКЕР #3: набрать 50 чел
              ↓
         Старт потока
```

**Три блокера:**
1. Утверждение программы с Ваней — без этого нельзя собирать контент
2. Готовый шаблон проекта — без него нет продукта для тестирования
3. 50 студентов — без них нет смысла запускать поток
