# AI-школа микрокурсов: переучить всех

> Идея: создать школу, которая производит пачками курсы-симуляторы, переучивающие людей работать с AI. Без воркшопов — видео + симулятор + AI-фидбек.

---

## Почему сейчас: данные

### Рынок взорвался за 3 месяца (январь–март 2026)

- **Cursor:** $2B ARR (март 2026), рост с $100M за год. 1 млн DAU. 60% выручки от корпоратов.
- **Claude Code:** $2.5B ARR (февраль 2026), удвоился с начала года. 29 млн установок в VS Code в день. 8 из 10 Fortune 10 — клиенты.
- **Lovable:** $400M ARR, 100 000 новых продуктов создаётся ежедневно.
- **Replit:** $265M ARR, рост 1556% год к году. 50M+ пользователей.
- **ChatGPT:** 910 млн еженедельных пользователей. 9M+ платящих бизнес-аккаунтов.

### Все роли меняются — данные

- **84%** разработчиков используют AI-инструменты, **47%** — ежедневно (Stack Overflow 2025)
- **Срок годности навыков** упал с 5–10 лет до **12–18 месяцев** (PwC)
- Навыки меняются на **66% быстрее** в AI-затронутых профессиях
- **56%** надбавка к зарплате за AI-навыки (PwC)
- **59%** руководителей компаний фиксируют AI skills gap (DataCamp/YouGov)
- **40%** мировой занятости затронуто AI (МВФ)
- Только **42%** сотрудников могут определить, где AI реально улучшит их работу

### Конкретные примеры трансформации

- **Klarna:** AI заменил 850 операторов поддержки. Выручка на сотрудника: $300K → $1.3M (+152%). Экономия $60M.
- **McKinsey Lilli:** 70% сотрудников используют, 500K запросов/мес, экономит 30% времени на ресерч.
- **BCG Deckster:** выполняет ~80% работы junior-аналитика (ресерч + слайды).
- **Shopify:** CEO приказал сотрудникам доказать, что AI не может делать их работу, прежде чем просить headcount.

### Главный инсайт: проблема не в инструментах, а в judgment

DataCamp 2026: видеокурсы (40% всего обучения) неэффективны — **23% руководителей говорят, что они не переводятся в реальные навыки**. Люди знают КАК использовать AI, но не знают КОГДА, ГДЕ и СТОИТ ЛИ.

**→ Это и есть рыночный gap, который мы закрываем симуляторами.**

## Продукт: как устроен курс-симулятор

### Формат: Видео → Задание → AI-фидбек → Repeat

```
┌─────────────────────────────────────────────────────────┐
│                    УРОК (15–20 мин)                      │
│                                                         │
│  1. Видео (5–7 мин)                                     │
│     Короткое видео: что делать и зачем.                  │
│     Показан реальный рабочий кейс.                       │
│                                                         │
│  2. Симулятор (8–12 мин)                                 │
│     Студент делает задание в реальном инструменте:       │
│     - Пишет промпт в Claude / ChatGPT                    │
│     - Работает в Excel / Google Sheets с AI              │
│     - Пишет текст / PRD / отчёт с AI-помощью            │
│     - Создаёт воронку / дашборд / автоматизацию          │
│                                                         │
│  3. AI-фидбек (мгновенный)                              │
│     Нейронка оценивает результат:                        │
│     - Что хорошо                                         │
│     - Что можно улучшить                                 │
│     - Показывает эталонное решение                       │
│     - Даёт балл (0–100)                                  │
│                                                         │
│  4. Redo или Next                                        │
│     Не прошёл 70% → переделай. Прошёл → следующий урок.  │
└─────────────────────────────────────────────────────────┘
```

### Типы симуляторов

| Тип | Как работает | Пример |
|-----|-------------|--------|
| **Prompt Lab** | Студент пишет промпт → AI выполняет → система оценивает качество результата | «Напиши промпт, который извлечёт из этого контракта все риски» |
| **Tool Sandbox** | Встроенный Excel/Sheets/CRM — студент работает с AI-помощником внутри | «Проанализируй эту воронку продаж с помощью Claude» |
| **Output Judge** | Студент загружает результат (текст, таблицу, презентацию) → AI оценивает по рубрике | «Напиши cold email с AI — мы оценим персонализацию, длину, CTA» |
| **Workflow Builder** | Студент собирает автоматизацию (n8n/Zapier-подобный интерфейс) → система проверяет логику | «Построй воронку: лид → обогащение → квалификация → письмо» |
| **Decision Sim** | Кейс-сценарий: AI предлагает 3 варианта, студент выбирает + обосновывает | «AI сгенерировал 3 варианта позиционирования — какой правильный и почему?» |

### Ключевое отличие от конкурентов

| | Видеокурсы (Udemy, Coursera) | Cohort-курсы (Maven) | GoPractice | **Наш симулятор** |
|-|-----|------|------|------|
| Формат | Видео + квиз | Live + проект | Симулятор решений | Видео + симулятор в реальных инструментах |
| Фидбек | Нет | Ментор | Автоматический (выбор из вариантов) | **AI-фидбек на открытые задания** |
| Инструменты | Не трогаешь | Иногда | Не трогаешь (теория) | **Работаешь прямо в инструменте** |
| Масштаб | ∞ | 30–50 чел. | ∞ | **∞** |
| Цена | $10–30 | $800–5000 | $200–400 | **$50–200** |
| Скорость создания | Месяцы | Месяцы | Месяцы | **Недели (AI-производство)** |

## 15 курсов: для каких ролей, чему учить

### Tier 1 — запускать первыми (самый массовый спрос + самый короткий цикл создания)

#### 1. «AI для Excel и Google Sheets»
**Для кого:** Все, кто работает с таблицами — аналитики, финансисты, маркетологи, менеджеры, бухгалтеры.
**Почему приоритет:** Самый массовый навык в мире (1.1B пользователей Excel). Курсы по Excel — исторически самые продаваемые. При этом «Excel + AI» — nascent категория, почти нет конкурентов.

**Чему учить (10 уроков, ~3 часа):**
1. Генерация формул промптом: «Напиши формулу, которая...» → AI пишет XLOOKUP/INDEX-MATCH
2. Анализ данных разговором: загрузить таблицу в Claude → задать вопросы → получить инсайты
3. Чистка и нормализация данных с AI
4. Автоматические сводные таблицы и дашборды
5. Прогнозирование: AI строит модель из исторических данных
6. Макросы и скрипты: AI пишет VBA/Apps Script по описанию
7. Когортный анализ и воронки с AI
8. Финансовое моделирование: P&L, cash flow, unit-экономика
9. Визуализация данных: AI подбирает тип графика и строит
10. Автоматизация отчётов: из данных → в готовый отчёт за 5 минут

**Цена:** $49–99 (или 5–10K ₽)
**Время создания:** 2–3 недели

## Производство курсов: как делать пачками

### Конвейер создания курса (2–3 недели на курс)

```
Неделя 1: Контент
├── День 1–2: Ресерч (AI)
│   AI анализирует: какие AI-инструменты используются в этой роли,
│   какие workflow меняются, что болит, что ищут в Google/YouTube.
│   Результат: структура курса из 8–12 уроков.
│
├── День 3–4: Сценарии уроков (AI + эксперт)
│   AI пишет черновики сценариев. Эксперт правит, добавляет кейсы.
│   Результат: 8–12 готовых сценариев.
│
└── День 5: Запись видео
    1 день записи: экран + talking head. Один оператор + Иван/эксперт.
    Монтаж: AI (Descript или аналог) + монтажёр за 2 дня.

Неделя 2: Симулятор
├── День 1–3: Создание заданий
│   Каждый урок = 1–3 задания.
│   AI генерирует варианты заданий, эксперт выбирает.
│   Настройка AI-проверки: рубрики, эталоны, промпты для оценки.
│
├── День 4–5: Тестирование
│   10 бета-тестеров проходят курс.
│   AI-фидбек калибруется по их результатам.
│
└── Сборка на платформе.

Неделя 3: Запуск
├── Лендинг (AI-генерация + дизайн)
├── Email-цепочка для запуска
├── Первые 100 студентов (из существующей базы / Telegram)
└── Сбор обратной связи, итерация
```

### Кто нужен для конвейера

| Роль | Кол-во | Что делает |
|------|--------|------------|
| **Продюсер курсов** | 1 | Управляет конвейером, приоритеты, дедлайны |
| **Эксперт по теме** | 1 на курс (парт-тайм) | Валидирует контент, записывает видео или консультирует |
| **AI-контент-инженер** | 1–2 | Генерирует сценарии, задания, рубрики с помощью AI |
| **Разработчик симулятора** | 1–2 | Платформа, интеграция AI-проверки, sandbox-ы |
| **Видеограф/монтажёр** | 1 | Запись + монтаж |

**Итого:** 5–7 человек = конвейер, производящий **2–3 курса в месяц**.

### Масштабирование: привлечение внешних экспертов

Для Tier 3 курсов (юристы, финансисты, HR) не нужно быть экспертом — нужно найти эксперта:
- Предложить revenue share (30–50% от продаж курса)
- Или фиксированную оплату за запись (200–500K ₽)
- Эксперт записывает видео + валидирует задания, всё остальное делает конвейер

**Источник экспертов:** выпускники наших курсов (785+ продактов, маркетологов, фаундеров).

## Маркетинг: как продавать пачками

### Канал 1: YouTube + органика (основной)

Формула: **1 курс = 5–10 YouTube-видео** (бесплатных) → каждое видео = вход в воронку.

Пример для «AI для Excel»:
- «Как я заменил 3 часа работы в Excel на 5 минут с Claude» → 1M просмотров
- «10 формул Excel, которые AI пишет за вас» → 500K просмотров
- «AI-дашборд из Excel за 10 минут — без навыков программирования» → 300K просмотров

Каждое видео → ссылка на бесплатный первый урок → upsell в полный курс.

### Канал 2: SEO/GEO — быть ответом на вопрос

- «Как использовать AI в Excel» — мы первые в выдаче

## Платформа: как технически

### Вариант 1: MVP (запуск за 2–4 недели)

- **Видео:** Kinescope / YouTube (unlisted)
- **Симулятор:** Google Forms / Typeform + API → Claude оценивает ответ → результат обратно студенту
- **LMS:** Stepik / Teachable / свой Next.js-сайт
- **AI-фидбек:** Claude API через webhook: студент отправляет ответ → промпт с рубрикой → оценка + фидбек

### Вариант 2: Кастомная платформа (через 2–3 месяца)

- Свой LMS с встроенными sandbox-ами:
  - **Excel sandbox:** Google Sheets iframe + Apps Script для отслеживания действий
  - **Prompt Lab:** текстовое поле → отправка в Claude API → оценка результата
  - **Code sandbox:** Monaco editor (как VS Code в браузере) → запуск → проверка
- AI-фидбек встроен в каждое задание
- Прогресс-бар, баллы, сертификаты
- Аналитика: где студенты застревают, какие задания переделывают

### Вариант 3: Killer feature — «AI-наставник»

- AI-бот в Telegram, привязанный к курсу
- Студент задаёт вопрос по теме → бот отвечает с контекстом курса
- Бот напоминает о незавершённых уроках
- Бот даёт дополнительные задания по слабым местам
- **Стоимость:** ~$0.01–0.05 на сообщение (Claude API) = пренебрежимо мало

## Почему это сработает: first principles

1. **Рынок гигантский и растёт.** AI в образовании: $7B → $137B к 2035 (CAGR 34.5%). Каждый knowledge worker на планете — потенциальный клиент.

2. **Навыки протухают за 12–18 месяцев.** Это не разовая покупка — это подписочная модель. Люди будут возвращаться за обновлениями.

3. **Производство дешёвое.** AI создаёт контент, AI проверяет задания, AI даёт фидбек. 200–400K ₽ на курс. Маржинальность 90%.

4. **Конкуренты не закрывают gap.** Видеокурсы не дают практики. Cohort-курсы не масштабируются. Симуляторы (GoPractice) учат решениям, а не инструментам. Мы — единственные, кто даёт «сделай сам в инструменте + получи AI-фидбек».

5. **Синергия с основным бизнесом.** Микрокурсы = массовый вход → upsell в КДП/Boost. B2B-канал для Руслана. Международный продукт без привязки к AJTBD.

6. **Иван перестаёт быть узким горлышком.** Курсы могут записывать приглашённые эксперты. Ивану не нужно преподавать — он задаёт стандарт и продюсирует.
