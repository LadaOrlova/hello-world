Дата: 2026-03-01
Версия: 1.0

## 1. Бизнес-модель

- **Тип:** Freemium SaaS (subscription)
- **Продукт:** AI-платформа "от идеи до работающего продукта"
- **Тиры:** Free ($0) / Pro ($29/мес) / Builder ($49/мес)
- **Целевой рынок:** Global, English-speaking, non-technical founders & solopreneurs

## 3. Расчёт LTV

### Формула для подписки:

```
LTV = ARPU × Gross Margin × (1 / Monthly Churn Rate)
```

### Расчёт:

```
LTV = $33 × 0.65 × (1 / 0.08) = $33 × 0.65 × 12.5 = $268
```

### LTV по тирам:

| Тир | ARPU | Gross Margin | Avg Lifetime | LTV |
|-----|------|-------------|-------------|-----|
| Pro ($29) | $29 | 65% | 10 мес (churn 10%) | $189 |
| Builder ($49) | $49 | 70% (ниже API costs per $) | 15 мес (churn 6.7%) | $515 |
| **Blended** | **$33** | **65%** | **12.5 мес** | **$268** |

## 4. Расчёт CAC по каналам

| Канал | Бюджет/мес | Visitors | CR visit→signup | Signups | CR signup→paid | Клиенты | CAC |
|-------|-----------|----------|----------------|---------|---------------|---------|-----|
| SEO/GEO (organic) | $500 (контент) | 5,000 | 5% | 250 | 1.5%* | 4 | $125 |
| Content (Twitter, Reddit, LinkedIn) | $300 (время) | 10,000 | 3% | 300 | 1.5%* | 5 | $60 |
| Google Ads | $3,000 | 3,000 | 8% | 240 | 1.5%* | 4 | $750 |
| YouTube | $500 (производство) | 15,000 | 2% | 300 | 1.5%* | 5 | $100 |
| Product Hunt (launch) | $200 (подготовка) | 5,000 | 10% | 500 | 1.5%* | 8 | $25 |
| **Итого (мес 1-3 avg)** | **$4,500** | **38,000** | **4.2%** | **1,590** | **1.5%** | **~24** | **$188** |

*CR signup→paid = 1.5% в первые 3 месяца (ниже целевых 3%, пока продукт формирует привычку)

**ВАЖНО:** CAC $188 при LTV $268 даёт LTV/CAC = 1.4 — это НЕ жизнеспособно для масштабирования. Нужно:
- Довести CR signup→paid до 3%+ → CAC ~$94 → LTV/CAC = 2.85
- Или снизить бюджет на ads до достижения конверсии → CAC organic ~$60-100

## 6. Payback Period

```
Payback = CAC / (ARPU × Gross Margin)
Payback = $188 / ($33 × 0.65) = $188 / $21.45 = 8.8 месяцев
```

**При целевом CAC ($90):**

```
Payback = $90 / $21.45 = 4.2 месяцев
```

**При целевом CAC и ARPU ($38, больше Builder):**

```
Payback = $90 / ($38 × 0.65) = $90 / $24.70 = 3.6 месяцев
```

## 7. Точка безубыточности

### Фиксированные расходы/мес:

| Статья | Стоимость |
|--------|----------|
| Команда (1-3 чел, зарплаты/opportunity cost) | $5,000-15,000 |
| Инфраструктура (Vercel Pro, Supabase Pro, PostHog) | $200-500 |
| Claude API (базовый) | $200-500 |
| Email (Resend) | $20-50 |
| Домен, SSL, мониторинг | $50-100 |
| **Итого (минимум, 1 чел)** | **$5,500** |
| **Итого (3 чел, полноценная команда)** | **$16,000** |

### Маржа с одного paid-клиента/мес:

```
Маржа = ARPU × Gross Margin = $33 × 0.65 = $21.45/мес
```

### Клиентов для безубыточности:

```
Break-even (1 чел) = $5,500 / $21.45 = 257 paid users
Break-even (3 чел) = $16,000 / $21.45 = 746 paid users
```

### При целевых 3% конверсии:

```
Для 257 paid: нужно 257 / 0.03 = 8,567 signups → ~171,340 visitors
Для 746 paid: нужно 746 / 0.03 = 24,867 signups → ~497,340 visitors
```

## 10. Эксперименты для валидации допущений

По принципу RAT (самые рискованные допущения — первые):

### Эксперимент 1: Готовность платить (Риск #2, Score 7.5)

**Допущение:** CR signup → paid > 3%
**Метод:** Smoke test landing page + fake paywall на MVP-0. Показать pricing после aha-момента, измерить клики на "Upgrade". Нет оплаты — есть intent.
**Стоимость:** $0 (только код)
**Срок:** 2 недели после MVP-0
**Метрика успеха:** > 5% кликов на "Upgrade" (intent > actual conversion)

### Эксперимент 2: Guided flow completion (Риск #1, Score 10.0)

**Допущение:** > 50% начавших flow доходят до деплоя
**Метод:** Запустить MVP-0 (idea → landing → deploy) на 100 пользователей, измерить completion rate per step.
**Стоимость:** $200-500 (paid ads для привлечения 100 users)
**Срок:** 2-3 недели после MVP-0
**Метрика успеха:** > 50% completion rate

### Эксперимент 3: CAC по organic каналам

**Допущение:** CAC < $90 через SEO + content
**Метод:** Опубликовать 3-5 SEO-статей + 2 Twitter threads + 1 Reddit пост. Измерить signups из organic за 30 дней.
**Стоимость:** $500 (время на контент)
**Срок:** 4-6 недель
**Метрика успеха:** > 100 signups из organic, implied CAC < $5/signup

### Эксперимент 4: Retention (Риск #6, Score 5.3)

**Допущение:** D7 retention > 30%, D30 > 15%
**Метод:** Когортный анализ первых 200 пользователей. Измерить: возвращаются ли, для чего (новый проект, итерация, валидация).
**Стоимость:** $0 (PostHog аналитика)
**Срок:** 30 дней после первой когорты
**Метрика успеха:** D7 > 30%, D30 > 15%

### Эксперимент 5: ARPU optimization

**Допущение:** Builder tier привлечёт > 30% paid users
**Метод:** A/B тест pricing page: вариант A (Pro + Builder) vs вариант B (только Pro). Измерить ARPU и conversion.
**Стоимость:** $0 (PostHog feature flags)
**Срок:** После 200+ paid users
**Метрика успеха:** Builder tier > 25% от paid, ARPU > $35

## 11. Критические метрики для мониторинга (Unit Economics Dashboard)

| Метрика | Порог тревоги | Действие при превышении |
|---------|-------------|----------------------|
| LTV/CAC < 2.0 | Жёлтый | Снизить paid ads, усилить organic |
| LTV/CAC < 1.0 | Красный | Остановить paid ads, пересмотреть модель |
| Monthly churn > 10% | Жёлтый | Анализ причин оттока, усилить retention |
| Monthly churn > 15% | Красный | Product problem, пересмотреть ценность |
| CAC > $150 | Жёлтый | Оптимизация конверсии, A/B тесты |
| Payback > 6 мес | Жёлтый | Увеличить ARPU или снизить CAC |
| Payback > 12 мес | Красный | Модель не работает |
| Gross margin < 50% | Красный | Оптимизация AI costs (кэширование, модели) |

## 12. Рекомендации

### Фаза 1 (месяц 0-3): Валидация экономики

- Фокус на organic каналы (SEO, content, PH launch) — CAC < $30
- НЕ масштабировать paid ads до LTV/CAC > 3.0
- Валидировать churn rate по первым когортам
- Тестировать pricing: $29 vs $39 vs $49 для Pro

### Фаза 2 (месяц 3-6): Оптимизация

- Достичь 257 paid users (break-even для 1 чел)
- Оптимизировать guided flow для конверсии > 3%
- Внедрить retention-механики (email nudges, next steps)
- A/B тест ценообразования

### Фаза 3 (месяц 6-12): Масштабирование

- Включить paid ads при LTV/CAC > 3.0
- Добавить годовую подписку (Pro: $249/год, Builder: $449/год) для снижения churn
- Рассмотреть one-time "Launch Pack" ($99-149) как fallback для Риска #2
- B2B enterprise tier для сегмента 4 (Builders в организации)
