Дата: 2026-03-01
Версия: 1.0

## Бизнес-модель
- **Тип:** Freemium -> SaaS (monthly/annual subscription)
- **Тарифы:** Free ($0) / Pro ($29/мес) / Builder ($49/мес)
- **Annual:** Pro $199/год, Builder $399/год (скидка ~30%)
- **Fallback:** Project Pack $19-29 (разовая покупка)

## Расчёт LTV

### Для подписки (основная модель):
- Average Lifetime = 1 / Monthly Churn = 1 / 0.07 = **14.3 месяцев**
- LTV = ARPU x Gross Margin x Average Lifetime
- LTV = $35 x 0.85 x 14.3 = **$425**

### Оптимистичный (churn 5%):
- LTV = $35 x 0.85 x 20 = **$595**

### Пессимистичный (churn 10%):
- LTV = $35 x 0.85 x 10 = **$298**

## Payback Period
- Organic: $56 / $29.75 = **1.9 месяцев** -- в норме
- Paid: $375 / $29.75 = **12.6 месяцев** -- нужна оптимизация
- Blended: $100 / $29.75 = **3.4 месяца** -- на границе

## Точка безубыточности

Фиксированные расходы/мес:
- AI API (Claude): $200-500 (зависит от пользователей)
- Хостинг (Vercel): $20
- Домен, email: $20
- Инструменты (PostHog free, etc): $50
- Время основателя: $0 (свой продукт)
- Маркетинг: $500-1,500
- **Итого: ~$800-2,000/мес**

Маржа с клиента/мес: $35 x 0.85 = **$29.75**

Клиентов для безубыточности:
- Минимум: $800 / $29.75 = **27 платящих клиентов**
- Максимум: $2,000 / $29.75 = **68 платящих клиентов**

При conversion rate 6%: нужно 450-1,130 signup'ов или 9,000-22,600 visitors.

## Эксперименты для валидации допущений

По RAT (самые рискованные допущения первые):

1. **CR signup -> paid (5-8%):** Landing page + free tier -> трекать conversion через PostHog. Минимум 200 signups для статистической значимости. Если <3% -- пересмотреть value proposition или paywall timing.

2. **Monthly churn (6-8%):** Первые 50 платящих клиентов x 2 месяца. Если >12% -- критичная проблема с retention, нужно пересмотреть "зачем возвращаться".

3. **CAC organic ($5-15):** 30 дней контент-маркетинга (10 статей + 5 Reddit-постов + 3 LinkedIn). Если organic CAC >$50 -- канал не масштабируется.

4. **ARPU $35:** A/B тест ценообразования. Варианты: $19/$39, $29/$49, $29/$79. Если ARPU <$20 -- пересмотреть тарифную сетку.

5. **WTP (есть ли готовность платить):** Smoke test: лендинг + paywall на Pro фичи. 500 visitors, если <2% кликают "Buy" -- гипотеза ценности не работает.
