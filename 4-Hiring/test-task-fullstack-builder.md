# Тестовое задание: Генератор гипотез сегментов с графами работ

## Контекст

На маркетплейсе Makers (makers-community.ru/marketplace) есть базовые AI-промпты для генерации сегментов и графов работ по методологии AURA. Можешь использовать их как основу — или сделать по-своему.

## Задача

Собери **работающий AI-инструмент**, который:

1. Принимает на вход описание продукта или бизнес-идеи
2. Ведёт диалог с пользователем — уточняет, задаёт вопросы, помогает сформулировать
3. Генерирует 5 гипотез сегментов (B2B или B2C) с обоснованием по AURA
4. Для каждого сегмента строит граф работ

## Что мы даём

- **Токен OpenRouter** с лимитом $5/день — используй для вызовов LLM в своём инструменте (токен получишь вместе с заданием)

## Формат сдачи

1. **Ссылка на работающий инструмент** (задеплоенный — Vercel, Netlify, что угодно)
2. **Loom-видео (до 5 мин):** покажи инструмент и расскажи, какие решения принял и почему
3. **Краткое описание:** какие решения принял и почему

## Ограничения

- **Время:** 2 дня
- **Не используется в коммерческих целях** — это только для оценки
