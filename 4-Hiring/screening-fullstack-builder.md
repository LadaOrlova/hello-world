## Пороги

- **48–60** — Сильный кандидат, рекомендовать на тестовое
- **36–47** — Средний, обсудить с командой
- **< 36** — Не проходит, отказ

## Красные флаги (немедленный отказ)

- Не вайб-кодит — пишет код только руками
- Узкий стек — только фронт или только бэк, не может закрыть целиком
- Не может показать построенный продукт
- Мотивация только деньгами
- Ждёт детального ТЗ — не готов к абстрактным задачам
- Не может отладить AI-сгенерированный код
