## Что может быть на следующих этапах

### Минимальный сценарий интерактивного онбординга через бот:

- [ ] Бот приветствует нового студента и показывает чеклист шагов
- [ ] Бот проверяет: привязан ли Telegram-аккаунт, подтверждён ли email, есть ли доступ к LMS
- [ ] Бот показывает только невыполненные шаги с кнопками для каждого действия
- [ ] По мере выполнения шагов бот обновляет чеклист
- [ ] Когда все шаги выполнены — бот поздравляет и даёт ссылку на первый урок

### Пользовательский опыт

- Бот напоминает о незавершённых шагах через N часов
- Контекстные подсказки: «Вот ссылка на чат вашей группы»

### Расширенные сценарии

- Настраиваемый чеклист: разные шаги для разных продуктов
- Аналитика: на каком шаге студенты чаще застревают
- Уведомление куратору, если студент не завершил онбординг за 48 часов
