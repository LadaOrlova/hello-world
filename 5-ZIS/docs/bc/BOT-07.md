## Минимальный сценарий адаптивных напоминаний:

- [ ] При онбординге студент выбирает формат прохождения: синхронный или асинхронный
- [ ] Бот отправляет напоминания перед занятием (синхронный формат) или при появлении новых материалов (асинхронный формат)
- [ ] Студент может изменить формат через команду боту
- [ ] Менеджер настраивает расписание напоминаний для потока в админке

### Пользовательский опыт

- Умные напоминания: бот учитывает часовой пояс студента
- Формат сообщений: краткая сводка + ссылка на урок

### Расширенные сценарии

- Цепочки напоминаний: серия сообщений перед важным дедлайном
- Адаптация к прогрессу: не напоминать об уроках, которые уже пройдены
- A/B-тестирование формулировок для повышения вовлечённости
