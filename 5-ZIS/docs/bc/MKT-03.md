# MKT-03: Рассылки — маркетолог может отправлять целевые рассылки по сегментам базы через email и Telegram

**Статус в прототипе:** Не реализовано

> **Источник:** Требования Лады (Владлены Орловой), маркетинг, 2026-03-13.

## Параметры реализации

### Критерии приёмки

- [ ] Маркетолог может создать рассылку с выбором сегмента (MKT-02) и канала (email / Telegram).
- [ ] Поддерживается email-рассылка: тема, текст, HTML-шаблон.
- [ ] Поддерживается Telegram-рассылка через бот: текст, кнопки, форматирование.
- [ ] Рассылка отправляется асинхронно (pg-boss job), с отслеживанием статуса (отправлено / ошибка / доставлено).
- [ ] Маркетолог видит статистику рассылки: отправлено, доставлено, открыто (email), ошибки.
- [ ] Есть защита от дублирования: повторная отправка одной рассылки одному пользователю не происходит.
