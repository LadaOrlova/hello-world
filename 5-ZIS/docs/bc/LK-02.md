# LK-02: В личном кабинете отображаются продукты, которые студент ещё не купил, с возможностью сразу перейти к покупке

**Статус в прототипе:** Работает

## Параметры реализации

### Критерии приёмки

- [ ] В личном кабинете `/student` отображается отдельный блок витрины «Доступно к покупке».
- [ ] В витрине показываются только продукты и тарифы, которые не входят в текущие доступы студента.
- [ ] Для каждой позиции витрины отображаются минимум: название продукта и доступные тарифы с ценами.
- [ ] Из витрины доступен прямой переход в сценарий покупки без выхода из личного кабинета.
- [ ] После приобретения продукта он исчезает из витрины «Доступно к покупке» и появляется в блоке «Мои доступы».
- [ ] При публикации нового доступного продукта в системе он появляется в витрине студента без ручной настройки для конкретного пользователя.

## Что работает в прототипе

Витрина доступных продуктов встроена в личный кабинет:

1. Студент открывает личный кабинет (`/student`)
2. Под блоком «Мои доступы» студент видит блок «Доступно к покупке»
3. В витрине отображаются только те продукты и тарифы, которые студент ещё не приобрёл
4. Для каждого продукта показано название и доступные тарифы с ценами
5. Студент может перейти к покупке прямо из личного кабинета
6. Когда в системе публикуется новый курс с открытым набором — он автоматически появляется в витрине
