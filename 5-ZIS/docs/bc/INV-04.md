# INV-04: Нумерация счетов и договоров формируется автоматически с учётом продукта (префиксы), года и порядкового номера — в настраиваемом формате

**Статус в прототипе:** Частично

## Параметры реализации

### Критерии приёмки

- [ ] При создании счёта, договора и акта номер документа формируется автоматически без ручного ввода.
- [ ] Для каждого типа документа используется единый формат нумерации, принятый в текущем прототипе.
- [ ] Система гарантирует уникальность номеров документов (без дублей) в рабочем контуре.
- [ ] Номер документа сохраняется в реестре и используется во всех связанных представлениях (кабинет компании, админка, PDF).
- [ ] Менеджер может создавать документы, не выполняя ручную нумерацию.
- [ ] Настраиваемые префиксы по продуктам в текущем прототипе зафиксированы как follow-up и не блокируют базовую приёмку автонумерации.

## Что работает в прототипе

Автоматическая нумерация документов:

1. Счета нумеруются автоматически по формату `INV-YYYYMMDD-XXXXXX`
2. Договоры нумеруются по формату `CTR-YYYYMMDD-XXXXXX`
3. Акты нумеруются по формату `ACT-YYYYMMDD-XXXXXX`
4. Нумерация сквозная — дубли исключены
5. Менеджеру не нужно вручную придумывать номер документа
