# LMS-14: Расписание и интеграция с календарём — студент видит расписание занятий и может добавить события в Google Calendar

**Статус в прототипе:** Не реализовано (заглушка в онбординге)

> **Источник:** Фидбек Леры (LMS/обучение), 2026-03-17.

## Параметры реализации

### MVP: Расписание + «Добавить в календарь» (без API)

#### Критерии приёмки

- [ ] В модели Cohort (поток) хранится структурированное расписание: список событий (дата, время, название, тип: лекция/семинар/дедлайн, ссылка на Zoom/Meet).
- [ ] Менеджер может редактировать расписание потока в админке.
- [ ] Студент видит расписание своего потока в LMS (отдельная вкладка или блок).
- [ ] Для каждого события доступна кнопка «Добавить в Google Calendar» (через URL-схему `calendar.google.com/event?...` — не требует API-интеграции).
- [ ] Ссылки на Zoom/Meet кликабельны и открываются напрямую.

> **Решение (2026-03-17):** MVP = расписание как структурированные данные в ZIS + кнопка «Добавить в календарь» через URL-схему (без OAuth, без API Google/Zoom). Полная API-интеграция — v2.

- [ ] Интеграция с Google Calendar API: автоматическое создание событий в календаре студента при зачислении.
