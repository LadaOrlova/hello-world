# PAY-12: Чеки возврата формируются и учитываются в системе при оформлении возврата средств

**Статус в прототипе:** Не реализовано

> **Источник:** Фидбек Насти (2026-03-11). «Чеки возврата — ЮКасса и Сплит формируют вроде бы автоматом, Т-банк видимо нет, так как возврат отправляем через банк (перевод по платёжке).»

## Параметры реализации

### Критерии приёмки

- [ ] При оформлении возврата через YooKassa чек возврата формируется автоматически на стороне провайдера и статус фиксируется в ZIS.
- [ ] При оформлении возврата через Сплит чек возврата формируется автоматически (аналогично YooKassa).
- [ ] При возврате через T-Bank (банковский перевод по платёжному поручению) менеджер вручную отмечает факт возврата в ZIS, чек коррекции формируется через онлайн-кассу (АТОЛ/аналог).
- [ ] В карточке заказа виден статус чека возврата (сформирован / не требуется / ошибка).
- [ ] Данные о чеках возврата доступны для финансовой аналитики.
