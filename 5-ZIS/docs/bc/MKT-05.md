# MKT-05: Реферальная программа — выпускник может пригласить друга и получить бонус при его покупке

**Статус в прототипе:** Не реализовано

> **Источник:** Требования Лады (Владлены Орловой), маркетинг, 2026-03-13.

## Параметры реализации

### Критерии приёмки

- [ ] Выпускник (alumni) получает персональную реферальную ссылку.
- [ ] При переходе по ссылке и покупке — реферал фиксируется в системе (кто привёл, кого привёл).
- [ ] Реферер получает бонус (скидку / денежное вознаграждение) — условия настраиваются.
- [ ] Реферал получает скидку на первую покупку — условия настраиваются.
- [ ] В админке видна статистика рефералов: кто привёл, сколько, конверсия, выплаты.
