# LMS-08: Обратная связь студента по материалам — студент может оставить реакцию и комментарий к уроку, а команда видит агрегированный фидбек

**Статус в прототипе:** Не реализовано

> **Источник:** Фидбек Леры (LMS/обучение), 2026-03-17.

## Параметры реализации

### Критерии приёмки (MVP)

- [ ] На странице урока в LMS студент может оставить реакцию: «полезно» / «не полезно».
- [ ] Опционально: студент может добавить текстовый комментарий к реакции (вопрос, замечание, благодарность).
- [ ] Реакция сохраняется в БД с привязкой к уроку, студенту и дате.
- [ ] В админке (и/или в кабинете куратора) видна агрегированная статистика фидбека по урокам: количество «полезно» / «не полезно», последние комментарии.
- [ ] Куратор может просмотреть комментарии студентов к урокам своей группы.
