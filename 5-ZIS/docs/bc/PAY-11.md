# PAY-11: Интеграция с T-Bank и Яндекс Сплит по API — формирование платёжных ссылок и обработка чеков без ручных действий менеджера

**Статус в прототипе:** Не реализовано

> **Источник:** Фидбек Насти (2026-03-11). «Давай проверим, можем ли мы формировать эти ссылки по API и целиком соединить этот процесс с чеками без ручных пометок.»

## Параметры реализации

### Критерии приёмки

- [ ] T-Bank подключен как платёжный провайдер через API — менеджер может сформировать ссылку на оплату без ручного создания в личном кабинете T-Bank.
- [ ] Яндекс Сплит подключен как платёжный провайдер через API — аналогично T-Bank.
- [ ] После создания ссылки система получает webhook об оплате и автоматически обновляет статус заказа.
- [ ] Кассовые чеки формируются автоматически при получении оплаты (связь с PAY-05).
- [ ] Все транзакции через T-Bank и Сплит отображаются в едином биллинге ZIS.
