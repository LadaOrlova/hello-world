## Что нужно реализовать в прототипе

Минимальный сценарий инвойсов для зарубежных компаний:

- [ ] Для зарубежных компаний формируется инвойс на **английском** или **двуязычном** (англ + рус) формате — чисто русский документ для иностранной компании не допускается (требование банковских проверок)
- [ ] Система формирует PDF-инвойс по шаблону американской компании
- [ ] Валюта инвойса: USD по умолчанию, EUR — по запросу. Другие валюты не поддерживаются
- [ ] Для РФ: используется счёт-договор (не инвойс) — на русском языке
- [ ] Для Армении: оплата принимается по русскому счёт-договору с ручным внесением реквизитов

### Что уже есть в платформе

- Компании и их реквизиты хранятся в системе
- Валюта документа (RUB/USD) уже поддерживается
