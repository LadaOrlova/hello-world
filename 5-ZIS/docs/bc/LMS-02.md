# LMS-02: Студент видит свой прогресс по курсу и может добавлять материалы в избранное и делать личные заметки

**Статус в прототипе:** Частично

## Параметры реализации

### Критерии приёмки

- [ ] В LMS студент видит структуру курса (модули и уроки) и свой текущий прогресс по прохождению.
- [ ] Студент может отмечать урок как пройденный, и статус прохождения сохраняется.
- [ ] По курсу отображается актуальное состояние прогресса (какие уроки пройдены, какие нет).
- [ ] Сводный прогресс по курсу отображается в личном кабинете студента.
- [ ] Студент может добавить урок в «Избранное» и видеть список сохранённых материалов.
- [ ] Студент может создавать и редактировать личные заметки к урокам.
- [ ] Личные заметки доступны только владельцу аккаунта (другие студенты, кураторы и менеджеры их не видят в рамках стандартного интерфейса).

## Что работает в прототипе

Отслеживание прогресса по курсу:

1. Студент открывает курс в LMS (`/student/lms/[enrollmentId]`)
2. Студент видит список модулей и уроков курса
3. После изучения урока студент нажимает «Изучено» — урок отмечается как пройденный
4. Студент видит, какие уроки уже пройдены, а какие ещё нет
5. Общий прогресс по курсу виден на главной странице личного кабинета
