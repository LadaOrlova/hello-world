## Что нужно реализовать в прототипе

Минимальный сценарий таргетированных уведомлений:

- [ ] Уведомления привязаны к конкретному продукту и потоку — студент получает сообщения только по своим курсам
- [ ] Студенты разных форматов (синхронный / асинхронный) получают разные наборы уведомлений
- [ ] Система не отправляет дубли: одно событие — одно уведомление
- [ ] Менеджер видит статистику отправленных уведомлений по студенту
