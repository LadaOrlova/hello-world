# PAY-08: Цены в иностранных валютах (USD и др.) автоматически пересчитываются по актуальному курсу ЦБ РФ, без необходимости ручного обновления

**Статус в прототипе:** Не реализовано

## Параметры реализации

### Критерии приёмки

- [ ] В системе настроен автоматический источник курсов ЦБ РФ и регулярное обновление курсов по расписанию.
- [ ] Для тарифов с базовой рублёвой ценой система автоматически рассчитывает цену в USD (и других поддержанных валютах) по актуальному курсу.
- [ ] Пересчитанные валютные цены сохраняются и используются в пользовательских витринах и лендингах без ручного обновления менеджером.
- [ ] В админке отображаются актуальный курс, дата и время последнего обновления и рассчитанные валютные цены по тарифам.
- [ ] При недоступности источника курса система не ломает checkout и использует последнюю валидную зафиксированную котировку с отметкой в логах.
- [ ] В заказе фиксируется валютная цена на момент оформления, чтобы последующие обновления курса не меняли уже созданный заказ.
- [ ] Ручная правка валютной цены менеджером (если используется) не удаляет возможность автоматического пересчёта по следующему циклу обновления.

## Что нужно реализовать в прототипе

Минимальный сценарий автоматического пересчёта валют:

- [ ] Система ежедневно получает актуальный курс валют с сайта ЦБ РФ
- [ ] Цены тарифов в иностранных валютах автоматически пересчитываются на основе рублёвой базовой цены
- [ ] Менеджер видит в админке текущий курс и пересчитанные цены
- [ ] На странице продукта студент видит цену в своей валюте

### Что уже есть в платформе

- Тарифы хранят цены в рублях и долларах — оба поля есть в базе данных
- На лендингах и в витрине отображаются цены в обеих валютах
- Менеджер может вручную изменить цену тарифа в админке

## Что может быть на следующих этапах

### Интеграции

- Автоматическая синхронизация курсов валют по расписанию (ежедневно)
- Поддержка нескольких валют (EUR, GBP и др.)

### Пользовательский опыт

- Автоматическое определение валюты по геолокации пользователя
- Отображение цены в локальной валюте на всех страницах

### Расширенные сценарии

- Фиксация курса на момент начала оформления заказа (защита от колебаний)
- Ручная корректировка курса менеджером при необходимости
- История изменений курсов для аналитики ценообразования
