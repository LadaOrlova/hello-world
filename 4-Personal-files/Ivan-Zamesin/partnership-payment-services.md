# Партнёрство: оплата зарубежных сервисов для студентов BOOST

## Кто мы

**BOOST** — интенсив по созданию продуктов с AI (вайб-кодинг) от Ивана Замесина. Недельный курс, где участники создают работающие продукты с помощью Claude Code. Удовлетворённость — 9.6/10, 95% участников без технического бэкграунда.

Потоки идут каждые 2–3 недели, группы по 50–100+ человек. Большинство студентов — из России.

## Проблема

Для обучения каждому студенту нужна подписка **Claude Code Max** — от $100/мес. Оплатить из России напрямую нельзя: нужна зарубежная карта + VPN. Существующие сервисы-посредники (типа Zaruba) берут высокую комиссию — 15–20%+, что при $100–200/мес ощутимо давит на студентов.

Сейчас мы вручную подсказываем студентам, как оплатить. Это отнимает время нашей команды, качество варьируется, и мы не контролируем процесс.

## Что нужно от партнёра

**Сервис оплаты зарубежных подписок**, который закроет следующие потребности:

1. **Адекватная комиссия** — ниже рыночной, потому что мы приводим лидов потоком
2. **Удобный интерфейс** — бот в Telegram или веб-интерфейс, через который студент сам всё оплатит за 5 минут
3. **Понятные способы пополнения** — рубли (карта, СБП), крипта, другие доступные варианты
4. **Поддержка студентов** — оперативная помощь, если что-то пошло не так (блокировка аккаунта, возврат, ошибка оплаты)
5. **Готовность к изменениям** — зарубежные сервисы регулярно меняют правила, блокируют аккаунты, требуют верификацию. Нужна команда, которая быстро адаптируется и находит рабочие способы
6. **Минимальная нагрузка на обе команды** — процесс должен работать максимально автономно

## Модель партнёрства

**Реферальная программа с накопительной комиссией:**

- Мы направляем студентов к партнёру через рекомендацию в курсе, гайды и чаты
- Комиссия сервиса для наших студентов — ниже стандартной (конкретный % — обсудим)
- **50% от комиссии** с каждого нашего студента идёт нам как реферальное вознаграждение
- Комиссия **накапливается всё время**, пока человек продолжает пользоваться сервисом — не только пока учится, а весь срок подписки на Claude Code и другие сервисы
- Прозрачная отчётность: сколько наших студентов, какой объём, какая комиссия

## Что мы даём

- **Стабильный поток клиентов:** каждые 2–3 недели новая когорта из 50–100+ человек, которым нужно оплатить Claude Code
- **Мотивированная аудитория:** студенты уже заплатили за курс 30–80 тыс. руб. — это платёжеспособные люди, которым оплата Claude Code нужна прямо сейчас
- **Интеграция в процесс обучения:** рекомендация партнёра встроена в гайды, FAQ, чаты — студент получает ссылку на сервис как часть курса
- **Долгосрочные пользователи:** после курса большинство продолжает использовать Claude Code для своих проектов — это не разовая оплата, а подписка на месяцы и годы

## Ответы на вопросы

**Какие именно подписки нужно оплачивать?**
В первую очередь — **Claude Max** ($100/мес или $200/мес). Также **ChatGPT** (от $20/мес до $200/мес, в зависимости от тарифа). Возможно в будущем Gemini, но это маловероятно. Основной объём — Claude Max за $100.

**Как студенты будут платить?**
В идеале — **Telegram-бот с минимальным фрикшном**: студент заходит, получает QR-код для оплаты через СБП или привязывает карту, деньги списываются автоматически. Никаких ручных переводов и переписок с менеджером.

**Это только на время курса или дольше?**
Это **долгосрочный сервис**, не привязанный к длительности курса. Студенты приходят через BOOST, но продолжают пользоваться подписками месяцы и годы для своих проектов. Мы рассчитываем, что человек будет оплачивать через сервис столько, сколько ему удобно — без ограничений по сроку.

**Что с подпиской после завершения курса?**
Подписка **остаётся у человека**. Курс заканчивается — подписка продолжает работать. Никаких аннуляций, отключений или ограничений.

**Мы хотим на этом зарабатывать?**
Да. Это не благотворительность — мы приводим платёжеспособных клиентов потоком и хотим получать реферальную комиссию за весь срок использования сервиса каждым студентом.
