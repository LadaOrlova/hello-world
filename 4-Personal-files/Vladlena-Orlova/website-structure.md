# Структура сайта zamesin.ru

---

## Карта сайта

```
zamesin.ru/
│
├── /                                    ← Главная (хаб экосистемы)
│
│   ── ПРОДУКТЫ ──
│
├── /producthowto/                       ← Тренинг "Как делать продукт"
│   ├── /producthowto/program            ← Программа тренинга
│   ├── /producthowto/pricing            ← Тарифы и оплата
│   ├── /producthowto/exam               ← Экзамен и сертификация
│   └── /producthowto/book/              ← Книга (главы)
│       └── /producthowto/book/{slug}    ← Отдельная глава
│
├── /boost/                              ← Интенсив BOOST
│   ├── /boost/program                   ← Программа интенсива
│   └── /boost/pricing                   ← Тарифы и оплата
│
├── /makers/                             ← Сообщество Makers
│
├── /for-teams/                          ← Корпоративное обучение (все продукты)
│
│   ── КОНТЕНТ ──
│
├── /cases/                              ← Кейсы выпускников
│   └── /cases/{slug}                    ← Отдельный кейс
│
├── /library/                            ← Библиотека знаний (статьи, гайды, переводы)
│   ├── /library/{slug}                  ← Отдельный материал
│   └── /library/tag/{tag}               ← Фильтр по тегу
│
├── /events/                             ← События (вебинары, запуски, конференции)
│   └── /events/{slug}                   ← Страница события
│
│   ── ОБЩЕЕ ──
│
├── /about                               ← Об авторе
│
└── /en/                                 ← Англоязычная версия
    ├── /en/course                       ← From Zero to First Product
    ├── /en/platform                     ← LaunchPilot / AURA
    ├── /en/library                      ← English articles
    └── /en/cases                        ← Case studies
```

### `/library/` — Библиотека знаний

**Почему "library", а не "blog":**
- "Блог" = личные заметки и мнения. "Библиотека" = систематизированное знание
- Позиционирует контент как ценный ресурс, а не поток постов
- Вмещает разные форматы: статьи, гайды, переводы, исследования
- Работает на бренд "Замесин = методология, наука, система"

**Альтернативы**, если "library" не нравится: `/knowledge`, `/research`, `/insights`

| Блок | Содержание |
|------|-----------|
| **Hero** | "Библиотека знаний по созданию продуктов" |
| **Типы контента** | Табы или фильтры: Статьи · Гайды · Переводы · Исследования |
| **Рубрики** | Теги (см. таблицу ниже) |
| **Сетка** | Карточка: обложка + заголовок + тип + тег + дата |

**Рубрики (теги):**

| Тег | Что внутри | SEO-запросы |
|-----|-----------|-------------|
| **JTBD** | Теория, разборы, "JTBD vs персоны", "Как формулировать работы" | jtbd, jobs to be done, jtbd интервью |
| **Продуктовый менеджмент** | PMF, unit economics, сегментация, стратегия | product market fit, сегментация клиентов |
| **Вайбкодинг** | "Что такое вайбкодинг", "MVP без программистов", AI-инструменты | вайбкодинг, mvp без кода |
| **Карьера** | "Как стать продактом", навыки, сертификация | стать продакт менеджером |
| **Исследования** | Переводы западных статей, обзоры трендов, данные | — |

**Типы контента:**

### `/events/` — События

**Почему "events", а не "webinars":**
- Вмещает всё: вебинары, запуски курсов, конференции, мастер-классы
- Не ограничивает формат, если появятся офлайн-события
- Одна страница для всех типов активностей

| Блок | Содержание |
|------|-----------|
| **Ближайшие** | Карточки предстоящих событий. Каждая: тип + тема + дата + время + CTA |
| **Архив** | Записи прошедших событий с email-гейтом |

**Типы событий:**

| Тип | Лейбл | Пример |
|-----|-------|--------|
| **Вебинар** | "Бесплатный вебинар" | "Новые принципы создания продуктов" |
| **Запуск** | "Открытый урок" / "День открытых дверей" | Запуск нового потока КДП или BOOST |
| **Конференция** | "Конференция" | Ежегодная конфа для выпускников |
| **Мастер-класс** | "Мастер-класс" | Разбор кейса / live-интервью |

**Карточка события:**

| Элемент | Содержание |
|---------|-----------|
| Тип | Лейбл с цветом: вебинар (синий), запуск (зелёный), конфа (фиолетовый) |
| Название | "Как найти PMF за 2 недели" |
| Дата и время | 20 марта, 19:00 МСК |
| Спикер | Ваня Замесин (+ гости, если есть) |
| CTA | "Зарегистрироваться" → сбор email |

**Страница события `/events/{slug}`:**

| Блок | Содержание |
|------|-----------|
| Hero | Название + дата + CTA регистрации |
| О чём | 3-5 пунктов: что узнаете |
| Спикер | Фото + био |
| Для кого | Кому полезно |
| CTA | "Зарегистрироваться" (до) / "Смотреть запись" (после — email-гейт) |

**Воронка:**
- До события: регистрация → email в базу → напоминание → live
- После: запись с email-гейтом → nurturing → продукт

**Правила перелинковки:**

| Откуда | Куда | Как |
|--------|------|-----|
| Библиотека (каждая статья) | КДП / BOOST / Makers | CTA-блок внизу: "Хочешь применить? →" |
| Библиотека | Кейсы | Упоминания в тексте: "Как это работает — кейс Rossko" |
| Библиотека | Книга | Ссылки на главы: "Подробнее в главе X" |
| Кейсы | КДП | CTA: "Хочешь так же?" |
| Кейсы | Библиотека | Ссылки на механику: "Что такое выход на предыдущую работу" |
| КДП | BOOST | Cross-sell: "Умеешь думать? Создай за неделю" |
| КДП | Makers | Cross-sell: "Продолжай расти в сообществе" |
| BOOST | КДП | Cross-sell: "Сначала разберись, ЧТО строить" |
| BOOST | Makers | Cross-sell: "Не останавливайся после интенсива" |
| Makers | КДП / BOOST | Upsell: "Пройди тренинг / интенсив" |
| События | КДП / BOOST / Makers | CTA после записи вебинара |
| Книга (бесплатные главы) | КДП | CTA: "Закрытые главы — для участников тренинга" |

## SEO-архитектура

### Приоритетные запросы

| Страница | Целевые запросы |
|----------|----------------|
| /producthowto | как делать продукт, тренинг jtbd, обучение продакт менеджменту, курс jtbd |
| /boost | вайбкодинг курс, создать mvp без программирования, AI создание продуктов |
| /makers | сообщество продактов, продуктовое сообщество |
| /for-teams | корпоративное обучение продакт менеджменту, тренинг jtbd для команды |
| /cases | кейсы jtbd, рост конверсии кейс, как увеличить выручку |
| /library | jtbd что это, jobs to be done на русском, сегментация клиентов, product market fit |
| /producthowto/book | книга jtbd, advanced jtbd книга |
| /about | ваня замесин, ivan zamesin |
| /events | вебинар jtbd, вебинар продуктовый менеджмент |

### Мета-теги (шаблон)

Каждая страница:
- `<title>` — до 60 символов, ключевое слово в начале
- `<meta description>` — до 160 символов, с CTA
- `<h1>` — один на страницу, уникальный
- Open Graph: title, description, image
- Schema.org: Course (тренинг), Article (библиотека), Review (отзывы), Event (события)

## Техническая реализация

| Параметр | Рекомендация |
|----------|-------------|
| **Фреймворк** | Next.js (уже в стеке) |
| **CMS для библиотеки** | MDX (файлы в репо) или Sanity (если нужен редактор для команды) |
| **Аналитика** | PostHog + GA4 |
| **Формы** | ZIS API (когда готов) / Tally на старте |
| **Хостинг** | Vercel |
| **Mobile** | Mobile-first, < 3 сек LCP |
| **Sticky CTA** | На всех продуктовых лендингах |
