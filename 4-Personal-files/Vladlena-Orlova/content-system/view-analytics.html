Источники данных что откуда берём
Я
Яндекс.Метрика
Посетители сайта, конверсия в заявку, UTM-разбивка по каналам, атрибуция first/last touch, понедельная динамика визитов
B
BotHelp
Подписчики бота + UTM (API), цепочки через вебхуки. Open/click rate — только дашборд, API не отдаёт
M
Mailganer
Open rate, click rate, unsub, подписчики и динамика — всё через API
Не покрыто этими источниками: клиенты, выручка, LTV, CAC, repeat purchase, когорты, мультитач-атрибуция покупок — нужна CRM или платёжная система. Подписчики YouTube — нужен YouTube Data API. Подписчики TG-канала (не бота) — нужен Telegram Bot API (getChatMemberCount).
