Март–апрель 2026 — сводка (все 3 сайта)

  • Визиты за март–апрель: ~42 000 по всем сайтам (КДП 19 700 + BOOST 14 000 + Блог 8 300)
  • BOOST — основной продукт: 2 674 визита на этой неделе, 41 цель, bounce 18.7%
  • Пик КДП: 16–22 марта (6 024 визита) — вебинар «Разбор продуктов»
  • Пик BOOST: 30 мар–5 апр (3 337) — старт 5-го потока
  • Блог стабилен: ~1 400–1 700 визитов в неделю, bounce 12–14% (самый низкий)
  • Цели за неделю: 73 (КДП 32 + BOOST 41)

  • UTM на TG: 41 переход за неделю, 63% — баннер на блоге

Что работает

  • Вебинар + follow-up рассылка дали 2x всплеск трафика на неделе 16–22 марта
  • Instagram показал лучшую конверсию по целям (2 из 5 визитов) — малый объём, но высокое качество
  • Яндекс (органика) — bounce rate 5%, лучший среди всех источников
  • Perplexity.ai — новый AI-источник с временем на сайте 3:55 (самый вовлечённый трафик)
  • Баннер на старом блоге — пассивно приводит 63% переходов на TG-канал (26 из 41)
  • TG-канал @zamesin: 17 постов за 40 дней, микс личного и продуктового контента

Что не хватает для полного отчёта

  • Лиды и клиенты — нужен AirTable
  • Выручка — нужна CRM / платёжная система
  • Email-метрики — нужен Mailganer API
  • Подписчики бот — нужен BotHelp API
  • TG-канал — посты и UTM-переходы подключены, нет подписчиков и просмотров (нужен TGStat API или ручной ввод)

Следующие шаги

  • Подключить Mailganer API для email-аналитики
