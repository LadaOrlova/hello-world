> Статус: черновик для обсуждения

---

## Проблема

Экосистема разрозненных сайтов:
- Старый блог Замесина (основной источник SEO-трафика)
- Новый блог/сайт с КДП ("Как делать продукт")
- AURA — отдельный сайт
- Отдельные лендинги: BOOST, экзамен, TeachEveryone

Нужно собрать всё в одну структуру, которая:
- Работает и для холодного трафика (из поиска), и для тёплого (из Telegram)
- Конвертирует на каждой странице
- Навигирует по сегментам (человек сразу видит себя)

## Раздел "Методология" (вместо "О нас")

Позиционирование: Замесин = автор методологии, а не "преподаватель курсов".

Содержание страницы:
- **AURA** — мета-фреймворк (что это, зачем, как связаны компоненты)
- **AJTBD** — методология исследований (отличие от обычного JTBD)
- **Кто за этим стоит** — блок про Замесина (часть истории методологии, не отдельная "О нас")
- **11 000+ выпускников** — social proof
- **Лид-магнит:** мини-диагностика AURA (7 вопросов) → оценка зрелости продукта за контакт

## Раздел "Кейсы"

### Фильтры

**По джобу (основной фильтр):**
| Фильтр | Job |
|---|---|
| Запуск продукта | Job 1 (25%) |
| Системный продакт-менеджмент | Job 2 (30%) |
| Рост и починка продукта | Job 3 (20%) |
| Карьера в продукте | Job 4 (15%) |
| Маркетинг и упаковка | Job 5 (10%) |

**По роли:**
- Собственник / предприниматель
- Product Manager
- Маркетолог / CMO
- Аналитик / проджект / дизайнер

**По отрасли (опционально, позже):**
- B2B SaaS
- B2C
- EdTech
- Финтех
- и т.д.

### Структура карточки кейса
- Фото + имя + роль + компания
- Цитата из "Точки А" (боль на входе)
- Что сделал на курсе
- Результат (цитата из "Точки B END")
- Оценка (NPS)
- Привязка к продукту: "Прошёл [название курса]"
- **Лид-магнит:** "Хочешь такой же результат? Забери чеклист [по джобу этого кейса]"

## Раздел "Календарь"

### Фильтры
- **Формат:** курс / вебинар / воркшоп / Q&A-сессия
- **По джобу:** для запуска / для системы / для починки / для карьеры / для маркетинга
- **Доступ:** бесплатно / платно / для корпоратов

### Карточка события
- Название + дата + время
- Формат (тег)
- "Подходит, если ваша задача — ..."
- Цена (или "бесплатно")
- Кнопка регистрации (лид-магнит: напоминание за контакт)

## Корпоративный раздел

### Офферы

**1. AI-воркшоп для команды (3 часа)**
- Преодолеть страх нейросетей
- Научиться вайб-кодить базовые вещи
- Онлайн или оффлайн, до 20-30 человек
- High-ticket, цена по запросу

**2. Лицензия на записи курсов**
- КДП, BOOST — для внутренней корп. академии
- Цена за лицензию

**3. Корпоративные когорты**
- КДП / BOOST для команды
- Кастомизация под задачи компании

**4. Индивидуальный консалтинг**
- High-ticket, по запросу

### На странице:
- Кейсы корпоративных клиентов
- Логотипы компаний
- Форма заявки

## Лид-магниты — распределение по сайту

| Где | Лид-магнит | За что отдаём контакт |
|---|---|---|
| **Главная** | Квиз "Какой курс подходит" | Результат квиза + рекомендация |
| **Главная** | Бесплатная вводная лекция | Доступ к записи |
| **Методология** | Шаблон графа работ / ABCDX-сегментации | Скачать шаблон |
| **Методология** | Мини-диагностика AURA (7 вопросов) | Оценка зрелости продукта |
| **Кейсы** (после прочтения) | Чеклист по джобу этого кейса | Скачать чеклист |
| **Страница продукта** | Бесплатный урок из курса | Доступ к уроку |
| **Знания / статья** | Связанный гайд или шаблон | Скачать материал |
| **Календарь** | Напоминание о событии | Email/Telegram подписка |
| **Сквозной баннер** | Ближайший бесплатный вебинар | Регистрация |

**Принцип:** каждая страница сайта = точка входа в воронку. Нигде человек не уходит с пустыми руками.

## Перелинковка

- Каждая статья в "Знаниях" → связанный кейс + продукт
- Каждый кейс → "хочешь так же? вот продукт"
- Каждый продукт → подборка статей + кейсов по теме
- Сквозной баннер на главный CTA (ближайший запуск или главный лид-магнит)
