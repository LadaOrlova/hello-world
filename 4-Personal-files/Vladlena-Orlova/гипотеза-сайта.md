## 4. Структура страниц — проваливание

Принцип: **ни одна страница не заканчивается тупиком**. Внизу каждой страницы — блоки, которые ведут человека дальше по сайту.

### Страница кейса (`/cases/:slug`)

| Блок | Содержание |
|---|---|
| **Кейс** | Имя, роль, компания, индустрия. Точка А → Что сделал → Точка Б (цитаты + цифры) |
| **CTA** | "Хотите такой же результат?" — варианты: разбор от Руслана / диагностика / записаться на обучение |
| ↓ Похожие кейсы | По джобу или индустрии |
| ↓ Статьи по теме | Ссылки в раздел "Знания" |
| ↓ Продукт | Курс, который прошёл герой кейса |

### Страница методологии (`/methodology`)

| Блок | Содержание |
|---|---|
| **AURA** | Что это, зачем, как связаны компоненты |
| **AJTBD** | Отличие от обычного JTBD |
| **Автор** | Замесин как часть истории методологии |
| **CTA** | Мини-тест "Проверь свой уровень" / диагностика от Руслана / вводная лекция |
| ↓ Кейсы | Кейсы, где методология дала результат |
| ↓ Статьи | Гайды и алгоритмы из базы знаний |
| ↓ AURA-продукт | Ссылка на `cpo-aura-framework.ru` |

### Страница из базы знаний (`/knowledge/:slug`)

| Блок | Содержание |
|---|---|
| **Статья / гайд / алгоритм** | Контент (фреймворк, инструкция, объяснение) |
| **CTA** | Скачать PDF-версию за контакт / записаться на диагностику |
| ↓ Кейсы по теме | Кейсы, где применялся этот фреймворк |
| ↓ Похожие статьи | Связанные гайды и алгоритмы |
| ↓ Курс | "Хотите применить на практике?" → страница курса |

### Страница курса (`/courses/:slug`)

| Блок | Содержание |
|---|---|
| **Курс** | Описание, формат, цена, для кого, результат |
| **CTA** | Записаться / оставить контакт (привязан к конкретному курсу → видно в CRM) |
| ↓ Кейсы выпускников | Кейсы по джобу этого курса |
| ↓ Статьи по теме | Гайды из базы знаний |
| ↓ Другие курсы | "Не уверены? Посмотрите другие форматы" |

### Страница календаря (`/events`)

| Блок | Содержание |
|---|---|
| **Список событий** | Фильтры: формат, джоб, бесплатно/платно |
| **CTA** | Подписка на мероприятия в Telegram-бота |

### Корпоративный раздел (`/corporate`)

| Блок | Содержание |
|---|---|
| **Офферы** | AI-воркшоп (3ч), лицензия на записи, корп. когорты, консалтинг |
| **Кейсы** | Корп. клиенты + логотипы |
| **CTA** | Форма заявки |

## 5. Курсы — навигатор по 5 джобам

> Источник: `2-Products/RU/NewKDP/jobs.md` (785 студентов, потоки 55-58)

| # | Фраза | Для кого | Формат | Цена | Результат |
|---|---|---|---|---|---|
| 1 | **Хочу запустить свой продукт** | Фаундеры, PM с идеей | 3 нед. | 60-90K | MVP с первыми клиентами |
| 2 | **Хочу перестать работать на чуйке** | PM Middle/Senior/Head | 6 нед. | 90-130K | Система решений + AI |
| 3 | **Продукт не растёт** | PM, собственник | 3 нед. | 60-80K | Диагноз + план роста |
| 4 | **Хочу войти в продакт** | Аналитики, проджекты, дизайнеры | 2 нед. | 35-50K | Кейс для портфолио |
| 5 | **Хочу упаковать продукт** | CMO, маркетологи | 2 нед. | 40-60K | Маркетинг от работ клиента |

+ **BOOST** (вайб-кодинг, 1 нед., от 29 900 ₽)
