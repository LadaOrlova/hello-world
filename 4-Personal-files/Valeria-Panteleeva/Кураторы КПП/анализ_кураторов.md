# Анализ работы кураторов: комментарии о студентах

> Дата анализа: 2 марта 2026
> Кураторы: Василина, Софья, Таня, Лиза
> Примерное количество студентов у каждого куратора: ~25–30 (у Софьи — 30)
> Важно: отсутствие записи о студенте = студент не отвечает / не идёт на контакт

## 2. Анализ по каждому куратору

### 2.1. Василина

**Формат работы:** собирает живые цитаты студентов, дополняет своими наблюдениями. Студенты раскрываются — рассказывают про личные обстоятельства (декрет, болезнь, выгорание, смену карьеры).

**Сильные стороны:**
- Глубокий эмоциональный контакт со студентами
- Студенты доверяют и делятся контекстом жизни
- Фиксирует качественные наблюдения ("отвечает сухо", "удивилась моим вопросам")
- Есть примеры конкретной помощи (помогла с вопросами)

**Слабые стороны:**
- Нет структуры в записях — сложно быстро оценить статус
- ~4–8 студентов не выходят на контакт — это существенная доля
- Не прослеживается системный подход к работе с "молчунами"

**Характерные трудности студентов:**
- Перегруженность на работе (массовый паттерн)
- Бюрократические ограничения (нет доступа к клиентам)
- Нет ресурса на применение знаний к своему продукту
- Сложный рекрутинг (особенно B2B)
- Экзистенциальные вопросы ("не знаю, останусь ли в профессии")
- Демотивация от внешних факторов (блокировки Telegram)

**Студенты в зоне риска:**
- Не посмотрела ни одной лекции, освободится после 8 марта
- Отвечает очень сухо, не смотрит воркшопы, не делает исследование
- "Если честно, ничего не успеваю"
- "Не знаю, останусь ли в профессии"
- Студент с "Жрутриент" — демотивирован результатами и блокировками
- ~4–8 студентов, с которыми нет контакта вообще

**Темп прохождения (из тех, с кем есть контакт):**

| Категория | Кол-во | Доля от записей |
|---|---|---|
| Идут в темпе или близко | ~7 | ~30% |
| Отстают, но стараются | ~10 | ~42% |
| Сильно отстают / не начинали | ~5–6 | ~25% |

---

### 2.2. Софья

**Формат работы:** краткие статусные заметки от третьего лица. Фиксирует факт ("идёт с потоком", "немного отстаёт"), но редко раскрывает детали.

**Сильные стороны:**
- Покрывает достаточно большую долю студентов (~23–24 из 30)
- Записи дают общее представление о темпе
- Фиксирует отдельные содержательные моменты (сложность восприятия, недовольство курсом)

**Слабые стороны:**
- Записи не содержат голоса студента — нет цитат, нет контекста
- Не видно действий куратора (помогла? направила? предложила?)
- 6–7 студентов из 30 не идут на контакт — каждый пятый
- Нет информации о конкретных трудностях у большинства студентов
- Запись "недоволен курсом" без раскрытия причины — критически важная информация потеряна

**Характерные трудности студентов (выявлено мало):**
- Сложно воспринимать информацию на воркшопах
- Информация заходит со скрипом
- Пересматривает воркшопы по несколько раз, тяжело
- "Очень путанно подаётся информация" (прямой негатив)
- Разница в часовых поясах

**Студенты в зоне риска:**
- "Недоволен курсом, проходит асинхронно" — причина не выяснена
- Студент, считающий подачу информации путанной
- Двое отстающих на 3-м модуле
- ~6–7 студентов без контакта

**Темп прохождения (из тех, с кем есть контакт):**

| Категория | Кол-во | Доля от записей |
|---|---|---|
| Идут с потоком | ~10 | ~43% |
| Немного отстают | ~9 | ~39% |
| Сильно отстают / проблемы | ~3–4 | ~15% |

### 2.3. Таня

**Формат работы:** структурированный шаблон с фиксированными полями (характер ответов, инициатива, модуль, темп, темы, трудности, настроение, комментарий куратора). Наиболее профессиональный подход к документированию.

**Сильные стороны:**
- Единственный куратор с системным форматом записей
- Фиксирует не только статус, но и свои действия (воодушевляла, разбирала продукт, предложила варианты наверстать)
- Выявляет конкретные содержательные трудности студентов
- Отмечает характер коммуникации и инициативу студента

**Слабые стороны:**
- Из ~25–30 студентов записи есть только о ~18 — **самая низкая доля контакта среди всех кураторов** (~7–12 студентов не на связи)
- Качество записей высокое, но охват — проблема
- Возможно, глубокая работа с каждым студентом забирает ресурс, и до части студентов руки не доходят

**Характерные трудности студентов (самая детальная диагностика):**
- Не понимает терминологию, делает задания на автомате
- Без реального продукта сложно придумывать джобы
- Хочет критерии оценки качества практики ("как понять, что я хорошо сделала?")
- Не может выстроить систему в голове
- Не хватает навыков коммуникации с респондентами
- Сложно переформулировать вопросы для респондентов
- Не понимает, как строить детализированный граф работ
- Сложность рекрутинга под бизнес-задачу

**Важное наблюдение:** ~10 из 18 студентов используют Таню **как справочник** по функциям курса, а не как ментора по содержанию. Роль куратора сводится к "вопрос-ответ по оргмоментам".

**Студенты в зоне риска:**
- Студент, игнорирующий чеки от личного куратора (общается только в рамках группы)
- Выгоревший студент (в начале хотел уйти в следующий поток)
- Студент, который досмотрит курс только в мае
- Студентка перед родами — объективно ограничена
- Демотивированная студентка без продукта
- ~7–12 студентов без контакта — **критическая зона**

**Темп прохождения (из тех, с кем есть контакт):**

| Категория | Кол-во | Доля от записей |
|---|---|---|
| Синхронно с потоком | ~10 | ~55% |
| Асинхронно / немного отстаёт | ~5 | ~28% |
| Сильно отстаёт | ~2–3 | ~15% |

### 2.4. Лиза

**Формат работы:** краткие заметки-впечатления. Доминирует формулировка "всё в порядке, отвечает кратко/редко, но дружелюбно". Фиксирует эмоциональный тон коммуникации, но не содержание обучения.

**Сильные стороны:**
- Самый высокий охват — есть записи практически по всем студентам
- Быстро считывает эмоциональный тон (дружелюбно, со смайликами, с реакциями)
- Умеет эскалировать сложные кейсы (советовалась с Лерой по токсичному кейсу)
- Замечает коммерчески ценные ситуации (отметила студента на отзыв)

**Слабые стороны:**
- Минимальная глубина записей — по большинству студентов невозможно понять реальный статус
- Не фиксирует текущий модуль, прогресс, конкретные трудности
- "Всё в порядке" повторяется у ~25 из ~33 записей — это не диагностика
- Почти нет содержательных вопросов к студентам
- "Не донимаю её" — сознательный отказ от контакта со студенткой, которая не отвечает на вопросы

**Характерные трудности студентов (выявлено крайне мало):**
- "В голове всё слегка сумбурно" (один студент)
- Нагрузка на работе x2 (один студент)
- Хотела перевестись — не рассчитала силы (одна студентка)
- Токсичное ожидание индивидуальных консультаций (один кейс)

**Студенты в зоне риска:**
- Студентка, хотевшая перевестись (пытается догнать, но масштаб отставания неясен)
- ~5–6 студентов, отвечающих "через раз" или "очень долго" — могут тихо уходить
- Студентка, которая не отвечает на вопросы куратора ("не донимаю её")
- Студент с нагрузкой x2 — "догонит на следующей неделе" (типичное откладывание)

**Темп прохождения (из записей):**

| Категория | Кол-во | Доля от записей |
|---|---|---|
| "Всё в порядке / успевает" | ~25 | ~75% |
| Немного отстаёт | ~4 | ~12% |
| Сильно отстаёт | ~2 | ~6% |
| Недавно присоединились | ~2 | ~6% |

Высокая доля "всё в порядке" — скорее всего следствие поверхностного контакта, а не реального благополучия.

## 3. Проблема неотвечающих студентов

| Куратор | Записей | Оценка без контакта | Доля без контакта |
|---|---|---|---|
| Василина | ~22–24 | ~4–8 | ~15–30% |
| Софья | ~23–24 из 30 | ~6–7 | ~20–23% |
| Таня | ~18 | ~7–12 | ~28–46% |
| Лиза | ~33 | ~0 | ~0% |

**Ключевые выводы:**

- **Таня** — при самом высоком качестве записей имеет самую низкую долю охвата. Глубина работы с каждым студентом, вероятно, забирает ресурс на охват. Либо её студенты менее контактны — но это маловероятно при случайном распределении.
- **Лиза** — наоборот: максимальный охват при минимальной глубине. Она дотягивается до всех, но контакт поверхностный.
- **Софья** — каждый пятый студент вне контакта при и так неглубоких записях.
- **Василина** — от 4 до 8 молчунов; при этом с остальными — глубокий контакт.

Неотвечающие студенты — это **скрытая красная зона**. Без контакта невозможно определить: студент тихо учится сам, или уже потерян.

## 4. Сквозные проблемы курса (видны у всех кураторов)

### 4.1. Перегруженность студентов работой
Самый частый барьер. Упоминается у всех четырёх кураторов. Студенты — работающие специалисты, курс конкурирует за их время с основной занятостью. Следствия: просмотр в записи, отставание, откладывание исследований.

### 4.2. Отсутствие своего продукта
Демотивирует и создаёт ощущение бесполезности практики. Студенты без продукта:
- Не могут придумать джобы "из головы"
- Чувствуют, что учатся "впустую"
- Практикуются на чужих продуктах (Яндекс Такси, Лавка) — лучше, чем ничего, но не даёт полного погружения.

### 4.3. Сложность материала и терминологии
- Воркшопы перенасыщены информацией
- Студенты пересматривают по несколько раз
- Есть прямой негатив: "путанно подаётся информация"
- Не хватает критериев качества: "как понять, что я хорошо сделала?"
- Графы работ, сегментация, рекрутинг — конкретные точки, где буксуют

### 4.4. Куратор как справочник, а не ментор
Особенно заметно у Тани и Лизы: студенты задают оргвопросы (как работает платформа, как попасть на воркшоп), но не обращаются за содержательной помощью. Возможные причины:
- Не понимают, чем куратор может помочь
- Стесняются задавать "глупые" вопросы
- Не привыкли к формату менторства

### 4.5. Групповые исследования — работающий якорь
Группы (Вешалки, Elektrolab и др.) удерживают студентов в процессе. Даже те, кто не тянет индивидуально, остаются через групповую динамику. Но: не все студенты попадают в группы, и некоторые используют группу как замену индивидуальной работе.

### 4.6. Воркшопы: ходить vs. смотреть в записи
Значительная часть студентов смотрит воркшопы в записи. При этом живое участие даёт больше — студенты сами это отмечают ("вопросы можно задать только сейчас"). Разрыв между живыми и асинхронными участниками может нарастать.

## 6. Рекомендации

### Критический приоритет

**Внедрить единый шаблон записей для всех кураторов.** За основу взять формат Тани, упростив для масштабируемости:

```
Студент: [имя]
Дата: [дд.мм]
Контакт: отвечает развёрнуто / по делу / кратко / не отвечает
Инициатива: да / иногда / нет
Модуль: [номер] / неизвестно
Темп: синхронно / немного отстаёт / сильно отстаёт / асинхронно
Трудности: [конкретно] / не обозначал
Настроение: спокойное / воодушевлён / тревожится / демотивирован
Действие куратора: [что сделал/а]
Красная зона: да / нет
```

### Высокий приоритет

- **Проработать проблему неотвечающих студентов.** Определить протокол: сколько попыток контакта, через какие каналы, когда эскалировать. Особенно актуально для Тани (~7–12 без контакта).
- **Дать кураторам скрипт содержательных вопросов.** Не "как дела?", а "на каком модуле ты сейчас?", "что было непонятно в последнем воркшопе?", "начал ли исследование?". Это поможет и глубине контакта, и полноте записей.
- **Провести калибровочную сессию.** Собрать 4 кураторов, разобрать 2–3 реальных кейса вместе, чтобы выровнять стандарт глубины и подхода.

### Средний приоритет

- **Определить маркеры "красной зоны" и протокол действий.** Комбинация из: не отвечает + не смотрит лекции + не ходит на воркшопы + не задаёт вопросов = эскалация.
- **Решить проблему "нет своего продукта".** Варианты: предложить учебные кейсы, активнее направлять в групповые исследования, дать список продуктов для практики.
- **Помочь Лизе углубить контакт.** Она дотягивается до всех — это ценно. Если добавить структуру и содержательные вопросы, может стать сильным куратором.
- **Помочь Тане с охватом.** Возможно, стоит разрешить ей вести короткие записи по "спокойным" студентам, чтобы оставить ресурс на глубокую работу с проблемными.

## 7. Общая оценка

Курс имеет **содержательно сильную базу** — даже отстающие студенты отмечают ценность материала. Основные проблемы лежат не в качестве контента, а в:

1. Несоответствии темпа курса реальным возможностям работающих людей
2. Отсутствии стандартизированной кураторской работы
3. Барьерах к практике (нет продукта, нет доступа к клиентам, сложный рекрутинг)
4. Недостаточном позиционировании роли куратора (студенты не понимают, чем куратор может помочь по содержанию)

При внедрении единого стандарта работы кураторов и решении проблемы неотвечающих студентов можно существенно снизить процент потерь и повысить качество прохождения курса.
