# Продажа КДП в записи: структура, тарифы, ревизия

*Итог: Решили проработать тариф с продажей КДП в записи, сохранив кураторскую поддержку и Q&A сессии. Необходимо провести ревизию материалов курса, определиться со структурой тарифа и ценообразованием, а также продумать формат парных воркшопов.*

**Ключевые моменты:**
•	Ранее была практика продажи доступа к текущему потоку КДП в записи для отдельных клиентов.
•	Текущий доступ к КДП в записи (58 поток) содержит лишнюю информацию (ссылки на чаты, устаревшие календари, упоминания сертификатов), что необходимо исправить при полноценной продаже тарифа.
•	В идеале воркшопы должны быть перезаписаны в студийном формате, а не представлены в виде сырой записи.
•	Ваня сможет перезаписать материалы с 18 апреля, необходимо предоставить ему бэклог задач.
•	Обсуждается структура тарифа в записи: возможно несколько вариантов, включая тариф с кураторской поддержкой и Q&A сессиями.
•	Предлагается запуск тестовой когорты (50 человек) для проверки гипотезы и определения оптимальной цены.
•	Обсуждается вопрос стоимости: текущая практика – 50-100 тысяч, предлагается рассмотреть диапазон 60-70 тысяч для тестового потока.
•	Важен формат проведения парных воркшопов в записи: либо полная переработка, либо сохранение парности с помощью кураторской поддержки и поиска партнеров для студентов.
•	Вариант: для парных воркшопов использовать заранее записанные разборы кейсов с представителями компаний с ссылкой на документ в Фигме.
•	Обсуждается возможность привлечения кураторов для проведения парных воркшопов и дачи обратной связи.
•	Предлагается объединить воркшоп по вайп-кодингу с курсом Boost.
•	Парные воркшопы в КДП: Advanced Jobs to be Done Interview, учимся создавать ценность, вайпкодинг (в группах), написание текста лендинга (условно парный).
•	Обсуждается возможность давать обратную связь (фидбек) куратором по домашкам в Фигме.
•	Возможно, можно ввести тариф "максимум" с фидбеком от Вани.
•	Решено изучить возможность полной интеграции вайпкодинга и КДП (перенаправление на курс Буст).
•	Определиться, что делать с воркшопом по вайпкодингу — один из самых геморных, и он не совсем про КДП.
•	Процент доходимости до воркшопов падает по мере продвижения по курсу.
•	Рассматривается возможность предоставления слотов времени с кураторами для проведения парных воркшопов.
•	Предлагается для асинхронного формата парный воркшоп либо убирать, либо проводить кураторам.
